# 申请材料与受理要求（研究稿）

## 一、本轮目标
在已有“使用场景分类”“授权类型与收费减免模式”研究的基础上，进一步向制度执行层推进，梳理博物馆图片授权工作中“申请材料与受理要求”这一关键环节。

本稿的重点不是设计一张最终表单，而是先明确：
- 受理环节究竟要判断什么；
- 为支持这些判断，申请人必须提交哪些基本信息与证明材料；
- 如何用制度表达把“受理”与“审核、收费、交付、责任追究”连接起来。

## 二、基本判断：申请材料不是为了“收表”，而是为了支撑四类管理判断
博物馆图片授权中的申请材料，不应只是形式化流程。其核心功能，是为以下四类判断提供依据：

1. **申请主体识别**
   - 申请人是谁；
   - 是否有合法主体资格；
   - 是否由被授权代表提出申请。

2. **使用目的识别**
   - 申请用途是什么；
   - 属于学术、教育、出版、宣传、商业还是其他场景；
   - 申报用途与实际用途是否一致。

3. **授权条件判断**
   - 拟使用图片的数量、对象、分辨率、范围、期限为何；
   - 是否涉及修改、再制作、再传播；
   - 是否需要附加声明、署名或特殊限制。

4. **收费与风险判断**
   - 是否属于减免对象；
   - 是否存在商业化、平台分发、赞助合作、品牌联名等情形；
   - 是否存在权利不清、超范围使用、内容敏感或其他管理风险。

因此，申请材料设计的好坏，直接决定制度能否真正落地。

## 三、建议的申请材料结构

### （一）主体信息类
用于确认申请主体身份、责任承担主体和联系渠道。

建议至少包括：
- 申请主体名称；
- 主体类型（个人 / 高校 / 研究机构 / 出版机构 / 媒体机构 / 企业 / 其他组织）；
- 联系人姓名与职务；
- 联系方式；
- 通讯地址；
- 统一社会信用代码或其他有效身份证明信息；
- 如为代理申请，需提交授权委托关系说明。

### （二）项目与用途说明类
用于判断使用场景、授权类型和收费适用规则。

建议至少包括：
- 项目名称；
- 使用目的；
- 使用场景分类；
- 成果形式（图书、论文、视频、展览、公众号、网站、课程等）；
- 是否营利；
- 是否有发行、销售、招商、赞助、商业合作或平台分发安排；
- 预计发布时间/展出时间/上线时间；
- 预计受众范围。

### （三）图片使用需求类
用于判断交付等级、范围限制与技术工作量。

建议至少包括：
- 拟申请图片对应的藏品名称、编号或其他识别信息；
- 拟申请图片数量；
- 所需文件规格（研究级、出版级、网络传播级等）；
- 使用媒介；
- 使用地域；
- 使用期限；
- 是否一次性使用或持续使用；
- 是否需高分辨率文件；
- 是否需特殊处理、重新扫描或重新拍摄。

### （四）修改与再利用说明类
用于识别最容易发生争议的使用行为。

建议明确要求申请人说明：
- 是否裁剪；
- 是否拼接；
- 是否局部放大；
- 是否叠加文字、动画、特效、包装设计元素；
- 是否用于封面、主视觉、广告画面；
- 是否会在多个平台重复传播；
- 是否存在转授权、再许可、素材库上传、二次分发等安排。

### （五）减免或特殊情形证明类
用于支撑收费与减免判断。

对于主张减免的申请，应要求补充：
- 学术或教育用途说明；
- 非营利性说明；
- 单位证明或项目立项说明；
- 论文录用通知、课程计划、展览合作函、研究计划书等支持材料；
- 如涉及合作方，应说明各方身份和责任关系。

### （六）承诺与签署类
用于把申请信息转化为可追责的制度凭据。

建议设置申请人确认条款，包括：
- 所填信息真实、准确、完整；
- 不得超出核准用途使用；
- 不得擅自转授权、转售、再分发；
- 应按要求署名并标注来源；
- 如用途、范围、媒介、期限发生变化，应重新申请或补充审批；
- 因虚假申报或违规使用产生后果的，由申请人承担责任。

## 四、受理要求的建议表达

### （一）先做形式受理，再做实质审核
制度上应区分：
1. **形式受理**：材料是否齐全、填写是否完整、基本要素是否明确；
2. **实质审核**：是否具备授权基础、是否属于可许可用途、适用何种条件和收费规则。

如此可避免“材料不全但已进入审批”或“受理与审批边界不清”的问题。

### （二）受理的最低必要要素应明确
建议规定：如申请材料缺少以下任一核心信息，可不予受理或要求补正：
- 申请主体信息；
- 具体用途说明；
- 拟使用图片识别信息；
- 使用媒介、范围和期限；
- 是否营利及相关商业安排；
- 联系方式和责任承诺。

### （三）应建立补正机制
受理制度不宜只有“通过/不通过”两种结果，还应允许：
- 一次性告知需补充或更正的材料；
- 设定补正期限；
- 补正后重新进入受理判断；
- 逾期未补正的，可视为撤回申请。

### （四）特殊情形应触发强化审查
如存在以下情况，制度上可要求进入强化受理或专门审核：
- 涉及商业广告、品牌营销、联名推广；
- 涉及大规模出版、平台分发或多渠道商业传播；
- 涉及敏感文物、限制公开信息或特殊合作协议；
- 涉及明显修改、再制作或高争议传播形式；
- 申请人与实际使用人不一致。

## 五、申请材料设计与后续制度章节的衔接关系

### （一）与“授权类型与基本条件”章节的衔接
申请材料中的“用途、范围、修改方式、传播方式”决定其更接近：
- 开放使用型；
- 附条件开放型；
- 受限许可型；
- 禁止/暂不授权型。

### （二）与“收费与减免规则”章节的衔接
申请材料中的“是否营利、是否商业合作、使用规模、分辨率需求、技术处理需求”直接影响是否收费以及收费层级。

### （三）与“交付方式与使用要求”章节的衔接
申请材料中的“使用媒介、展示尺寸、输出形式、版本要求”将决定交付文件的规格与后续使用限制。

### （四）与“风险控制与责任承担”章节的衔接
申请材料中的承诺条款、用途说明、主体信息，是后续认定超范围使用、虚假申报、违规传播责任的重要依据。

## 六、可直接转化为制度条文的表达方向
后续草案中，“申请材料与受理要求”章节可考虑写成以下结构：
1. 申请主体应提交基本身份信息；
2. 申请人应明确说明使用目的、使用方式、使用范围和使用期限；
3. 申请人应准确标明拟申请图片对象及文件需求；
4. 对主张减免、公益、学术、教育等特殊情形的，应提交相应证明材料；
5. 申请材料不完整或关键要素不明确的，可要求补正；
6. 对存在高风险或复杂情形的，可转入专门审查程序；
7. 申请人应对申报内容真实性和后续合规使用承担责任。

## 七、本轮阶段性结论
本轮研究形成三个可直接用于后续起草的判断：
1. 申请材料制度的核心不是“收集信息”，而是支撑主体识别、用途识别、授权条件判断和收费/风险判断四类管理任务；
2. 受理应与审核区分，形成“形式受理—补正—实质审核”的递进结构；
3. 申请表字段设计必须与授权类型、收费规则、交付要求和责任追究机制联动，否则制度将出现表单与审批脱节的问题。
