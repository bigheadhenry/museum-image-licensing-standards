# 图片服务办理说明（示范稿）

> 说明：本示范稿用于博物馆官方网站、数字资源平台或图片服务页面中的“服务办理说明”文本。其重点不是解释授权是否准许，而是说明：在授权申请或开放资源使用之外，馆方可提供哪些图片相关服务、哪些服务可能收费、哪些服务需要预约或进一步审核、交付通常如何进行，以及申请人应如何理解“许可判断”与“服务提供”之间的关系。正式实施时，可结合馆方平台结构、服务能力、收费政策和技术条件调整。

## 一、本说明适用于哪些事项？
本说明主要适用于下列与图片资源相关的服务事项：
1. 高分辨率图片文件提供；
2. 出版级或印刷级文件处理；
3. 文件格式转换、尺寸调整、色彩处理等技术性服务；
4. 指定渠道交付、下载链接管理、项目包整理等交付服务；
5. 加急处理、专项说明、补充文书等配套服务；
6. 其他由本馆根据资源条件和业务能力提供的图片服务事项。

## 二、图片服务与图片授权是什么关系？
图片授权与图片服务不是同一件事。

- **图片授权**解决的是：某项使用是否被允许、在什么范围内被允许；
- **图片服务**解决的是：馆方是否提供文件、提供什么规格、以何种方式交付、是否需要额外处理以及是否收取相应服务成本。

因此：
1. 获得授权，并不当然意味着可取得最高规格文件或所有相关服务；
2. 提供文件服务，也不当然意味着使用者获得超出核准范围的使用权；
3. 开放资源可降低申请门槛，但并不当然包含所有高规格文件、技术处理或加急支持。

## 三、馆方通常可提供哪些图片服务？
根据实际业务能力，馆方可提供的图片服务通常包括：

### （一）基础文件提供服务
- 现有公开版本下载；
- 已授权事项对应文件交付；
- 标准分辨率文件提供；
- 指定接收渠道的文件发送。

### （二）规格升级服务
- 更高分辨率文件；
- 出版级、印刷级或高清展示级文件；
- 批量文件整理与打包；
- 多版本文件组合交付。

### （三）技术处理服务
- 文件格式转换；
- 尺寸或分辨率调整；
- 基础色彩处理或输出适配；
- 文件重命名、按项目整理或清单匹配；
- 其他合理范围内的技术性处理。

### （四）专项支持服务
- 加急处理；
- 纸质证明、补充说明或附加文书；
- 特殊项目交付安排；
- 其他需要单独协商的服务事项。

## 四、哪些服务可能需要单独判断或另行收费？
以下事项通常不宜当然视为基础服务，馆方可根据工作量、风险和资源条件另行判断是否提供，并视情收取技术/服务处理费：
1. 高分辨率、出版级、印刷级文件；
2. 批量图片处理；
3. 文件格式批量转换或特殊输出适配；
4. 加急处理；
5. 补开纸质证明、情况说明、附加文书；
6. 特殊整理、定向交付或项目化服务；
7. 其他明显超出一般下载或标准交付范围的事项。

## 五、哪些情况下馆方可能无法提供相关服务？
存在以下情形之一的，馆方可不提供或暂缓提供相关服务：
1. 相关资源权利基础不清或尚未完成授权判断；
2. 使用用途尚未获批或条件未明确；
3. 文件规格需求明显超出馆方现有保存、处理或交付能力；
4. 申请人尚未完成收费确认、协议签署或其他前置程序；
5. 所申请事项涉及较高风险、敏感内容或后续控制难度较高；
6. 其他依法依规或根据馆方业务条件不宜提供服务的情形。

## 六、如何理解文件规格与费用关系？
文件规格、处理深度和交付要求，通常会直接影响是否收取技术/服务处理费。一般而言：
- 公开页面已提供的基础版本，可能可直接下载或低成本获取；
- 较高规格文件、批量整理、专项处理和加急服务，通常需要另行判断工作量与服务成本；
- 减免许可使用费，并不当然意味着所有文件服务都免费；
- 仅收技术/服务处理费，也不代表该事项属于无需授权判断的自由使用。

## 七、服务办理通常需要经过哪些环节？
图片服务事项通常可按照以下链条办理：
1. 服务需求提出；
2. 与授权状态或开放规则进行匹配判断；
3. 确认文件规格、数量、用途和交付方式；
4. 确认是否收费、是否适用减免或仅收服务费；
5. 完成必要确认或支付；
6. 办理文件交付；
7. 形成交付记录并保留必要留痕。

对复杂事项，馆方可要求先完成授权审批，再进入服务办理。

## 八、交付方式通常有哪些？
根据事项风险和文件规格，馆方可采用以下一种或多种方式交付：
1. 在线下载链接；
2. 系统账号下载；
3. 指定邮箱发送；
4. 项目包集中交付；
5. 受控访问或限时下载；
6. 其他馆方认为适当的交付方式。

对高风险或高规格文件，馆方可设置下载期限、接收主体限制、次数限制或其他控制措施。

## 九、申请人取得文件后还应注意什么？
取得图片文件后，申请人仍应遵守相应授权条件、开放规则和交付说明，尤其应注意：
1. 仅在核准用途、范围、期限、平台和媒介内使用；
2. 按要求署名并附必要声明；
3. 不得因获得文件而当然主张取得再次授权、转售、再分发或无限制使用权；
4. 不得擅自上传至素材平台、开放下载平台或第三方资源库；
5. 不得超出获准范围进行再制作、广告化包装或高敏传播。

## 十、加急、专项处理和批量事项如何看待？
加急、专项处理、批量文件整理等事项，通常需要馆方结合实际工作量、当前业务安排和风险情况单独判断。申请人如确有时间要求或特殊需求，建议在申请时一并说明：
- 项目时间节点；
- 需要的文件规格和数量；
- 是否需要加急；
- 是否存在多平台、多版本或复杂交付安排；
- 是否需要附加说明或特殊格式输出。

馆方可根据情况决定是否提供、如何安排以及是否收取相应服务费用。

## 十一、哪些事项建议尽早沟通？
以下事项建议申请人尽早与馆方沟通，而不要等到项目临近时再提出：
1. 大批量图片使用；
2. 出版级/印刷级高清文件需求；
3. 多语种、多平台或跨境传播项目；
4. 展陈、视频、广告、品牌合作等复杂项目；
5. 对时间要求极高的项目；
6. 需要重新拍摄、重新扫描或较大工作量技术支持的项目。
