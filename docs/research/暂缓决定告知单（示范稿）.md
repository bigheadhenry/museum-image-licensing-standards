# 暂缓决定告知单（示范稿）

> 说明：本示范稿用于在图片授权申请已经进入审核或审批判断阶段，但现阶段材料、事实、风险评估或内部确认条件尚不足以支持馆方立即作出“同意 / 附条件同意 / 不予同意”等最终决定时，向申请主体发出正式书面告知。该文书与《授权申请被暂缓、限制或不予同意情形说明（示范稿）》《授权申请受理、审核与审批工作流程说明（示范稿）》《审核意见单（示范稿）》《补正通知单（示范稿）》等配合使用，用于把“暂缓决定”从对外说明层与流程层进一步落实到正式执行文书层。

## 二、暂缓决定事项
经对贵方提交的图片授权申请及相关补充材料进行审核，现就以下事项作出**暂缓决定**：
- 当前暂不作出同意决定；
- 当前暂不作出附条件同意决定；
- 当前暂不作出不予同意决定；
- 待补充核实、补充说明、进一步评估或完成内部确认后，再进入下一判断。

## 三、暂缓原因
本次暂缓决定，通常系基于以下一种或多种情形（可勾选或填写）：
- 当前关键事实仍不够明确，暂不足以形成最终授权判断；
- 申请主体、实际使用主体、发布主体、合作方之间关系仍需进一步核实；
- 使用目的、平台、传播范围、期限、语种、地域或合作安排仍需进一步说明；
- 涉及封面/主视觉、广告传播、联名合作、平台招商、跨境传播、多语言发布等高风险用途，需进一步评估；
- 涉及敏感图片、权利状态复杂资源、第三方权利或其他需专项确认事项；
- 已提交样稿、方案或说明材料，但仍不足以支持明确决定；
- 需内部联合复核、专项会审、管理层确认或法务支持；
- 需等待相关外部条件、项目安排、合作文件或证明材料进一步明确；
- 其他：

## 四、馆方当前判断说明
现阶段馆方的判断并非当然否定贵方申请，而是认为：
1. 当前申请已超出“仅靠一般材料即可直接作出决定”的状态；
2. 若在当前事实基础上直接作出最终决定，可能导致授权边界不清、风险控制不足或后续执行困难；
3. 因此，需先将有关事实、条件、范围、样稿、合作关系或风险判断进一步澄清后，再决定是否同意、附条件同意、缩小范围后同意，或不予同意。

## 五、暂缓期间的处理要求
根据事项情况，贵方可按馆方要求继续配合以下一种或多种工作：

### （一）补充说明或补充材料
- 补充主体关系说明；
- 补充传播方案、页面样稿、版式示意、视频脚本摘要或视觉方案；
- 补充合作协议摘要、平台分发安排、联名合作说明或项目证明；
- 补充样图、页面截图、上线计划或传播节奏说明；
- 补充权利来源、授权链条或第三方权利处理说明；
- 其他：

### （二）调整申请方案
- 缩小拟申请图片范围；
- 缩小使用平台、使用媒介、地域、语种或传播范围；
- 调整成果形式、展示方式、修改方式或发布计划；
- 取消高风险安排（如广告投放、联名推广、封面/主视觉、平台再分发等）；
- 其他：

### （三）等待进一步确认
- 等待馆内专项会审 / 联合复核；
- 等待管理层确认；
- 等待法务或合作关系核验；
- 等待贵方项目安排或外部证明材料明确；
- 其他：

## 六、下一步可能的办理路径
在暂缓期间事项澄清后，馆方可根据情况进入以下一种或多种后续路径：
1. 补充说明后恢复审核并作出正式决定；
2. 调整申请范围后进入附条件同意路径；
3. 转入强化审查路径；
4. 经补充后作出不同意决定；
5. 视情要求贵方重新提交新的完整申请；
6. 其他：

## 八、特别说明
1. 暂缓决定并不当然等于不予同意，也不当然意味着后续一定会获批；
2. 暂缓决定的主要功能，是在风险尚未可控、事实尚未充分清楚时，为事项保留进一步判断空间；
3. 暂缓期间，申请事项原则上不视为已获授权，申请人不得以“正在办理中”“馆方正在研究”为由擅自先行使用相关图片；
4. 如后续补充情况显示事项风险仍高、条件仍不明确或明显超出馆方管理原则，馆方仍可作出不予同意、限制申请范围或转入更严格路径的决定。

## 十一、使用说明（馆内）
1. 本告知单适用于“当前尚不足以形成最终决定，但仍保留继续判断空间”的事项；
2. 如事项仅属一般材料不足，原则上可优先使用《补正通知单》承接；
3. 如事项已明显达到不予同意条件，原则上不应长期以暂缓决定替代正式不同意决定；
4. 暂缓决定应尽量写明暂缓原因、后续要求和下一步可能路径，避免形成事实上的无限期搁置；
5. 本告知单应与申请编号、审核意见、后续补充材料、审批结论和留档材料对应保存。
