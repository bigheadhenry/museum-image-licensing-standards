# 国内博物馆图片使用政策与版权声明比较（第二轮）

## 一、本轮目标
在第一轮已经完成故宫博物院、中国国家博物馆等对象的正文核验，并对上海博物馆、南京博物院、湖南博物院建立“官方入口待补证”状态之后，本轮继续推进国内案例补证，重点不在于夸大已经掌握的内容，而在于：
- 尽量把“官方入口是否真实存在”与“正文是否已稳定抽出”区分清楚；
- 对已可确认的国内馆站点结构、版权入口和页脚权利归属信息进行补证；
- 为后续形成更稳健的国内比较方法和制度原型提供第二轮基础。

## 二、本轮补证对象与结果

### 1. 上海博物馆（官方入口进一步确认，但正文仍未稳定抽出）
本轮核验页面：
- `https://www.shanghaimuseum.net/mu/frontend/pg/copyright`
- `https://www.shanghaimuseum.net/mu/frontend/pg/privacy`

本轮可确认要点：
1. 上述两个路径均可稳定返回 HTTP 200；
2. 页面源码和可见结构中，能够稳定确认站内存在“隐私条款”“版权声明”两个官方信息公开入口；
3. 页面底部可稳定看到“Copyright © 2015 ShanghaiMuseum.net 上海博物馆 版权所有”等权利归属表达；
4. 页面导航中还能看到“数字文物库”“典藏精选”“三维藏品”“AR藏品”“网展”“视频”“全景展厅”“海报”“桌面”“手册”“馆刊”“图录”等数字资源相关栏目。

本轮仍未能稳定确认的事项：
1. 当前自动提取与源码清洗结果，仍未抽出独立、完整的版权声明正文；
2. 因此尚不能负责任地断言其具体条款内容，如是否明确规定商业使用、转载、修改、来源标注等细则。

本轮阶段性判断：
- 上海博物馆已不只是“疑似存在版权页面”，而是可以更稳地归入“已确认官方版权/隐私入口存在”；
- 但就证据等级而言，仍应保持为“官方入口已确认、正文待补提取”，而不应误记为“正文已核验”。

制度启发：
- 国内馆在网站层面往往已形成“版权声明/隐私条款 + 数字资源栏目”的公开框架；
- 即使当前正文抓取受限，这种结构本身也说明：国内数字资源使用治理，常常与网站治理、平台内容管理和数字资源展示栏目并列存在。

### 2. 南京博物院（官方版权/隐私路径继续确认，正文仍未稳定抽出）
本轮核验页面：
- `https://www.njmuseum.org.cn/zh-CN/copyright`
- `https://www.njmuseum.org.cn/zh-CN/privacy`

本轮可确认要点：
1. 版权与隐私路径可稳定返回 HTTP 200；
2. 域名已稳定表现为 `njmuseum.org.cn` 官方站点结构的一部分；
3. 页面本体明显采用前端框架渲染，静态抽取结果仍较弱，但可确认其并非随机报错页或失效路径。

本轮仍未能稳定确认的事项：
1. 尚未抽取到可直接引用的版权声明正文；
2. 尚不能确认其是否公开列出图片使用、转载、商业使用或联系路径等条款。

本轮阶段性判断：
- 南京博物院可继续维持“已确认官方入口但正文待补提取”的证据状态；
- 与上海博物馆类似，后续如要提升证据等级，仍需借助更强的页面渲染或源码定位方法。

制度启发：
- 对国内案例研究，单纯依赖一般文本抽取往往不足；
- 需要接受“先确认官方入口真实存在，再逐步提升正文证据”的分层方法，而不是把所有对象都强行处理为已核验正文。

### 3. 陕西历史博物馆（补到官网性质说明页与页脚权利归属）
本轮核验页面：
- `https://www.sxhm.com/about/detail/209.html`

本轮可确认要点：
1. 页面明确表述“陕西历史博物馆官方网站，是我们与公众进行跨时空交流对话的重要平台”；
2. 文本强调官网是与公众互动、增进交流、推动公众走近博物馆的重要线上窗口；
3. 页面页脚可稳定确认“Copyright 2021 陕西历史博物馆版权所有 All Rights Reserved.”；
4. 这说明陕历博至少在官网整体层面具有明确的权利归属表达。

本轮未确认的事项：
1. 尚未定位到单独的图片授权页、版权声明页或图片使用规则页正文；
2. 目前仍不能确认其是否公开区分一般网站内容规则与图片个案授权路径。

本轮阶段性判断：
- 陕西历史博物馆目前可作为“已确认官网窗口定位 + 已确认版权所有页脚信息，但未定位到专门图片使用规则页”的对象纳入线索池；
- 相比上海博物馆、南京博物院，它的价值不在于已拿到完整规则，而在于补强了“官网平台属性 + 权利归属表达”这一证据层。

制度启发：
- 国内馆的官网说明页本身，有时也承担“平台定位说明”功能；
- 在制度设计层面，这提示我们：外部规则体系并不只包括版权声明和授权说明，也可能包括对官网/平台性质的公开表述。

## 三、本轮未完成但值得记录的问题

### 1. 湖南博物院
本轮尝试直连官网及常见版权/隐私路径时，抓取过程中出现 SSL EOF 类错误，未能形成稳定页面正文或路径证据。

当前判断：
- 湖南博物院仍应保留在“待继续补证的重要对象”中；
- 后续可改用浏览器模拟、站内搜索、备用网址或人工定位菜单路径方式继续补证。

## 四、本轮比较后的国内阶段性判断

### 1. “官方入口存在”本身已是有价值的证据层
在国内案例中，公开规则往往不像国外那样集中在单一的 Open Access / Image Licensing 页面中。很多时候，先能确认：
- 是否存在官方版权声明入口；
- 是否存在隐私条款入口；
- 是否存在数字资源/数字文物库/图录/三维藏品等数字内容栏目；
本身就已经能反映机构的公开治理结构。

### 2. 国内案例研究更需要证据分层，而不是一次性下结论
经过两轮补证后，可以更明确地看到：
- 有些对象可拿到正文（如故宫、国博）；
- 有些对象只能先确认官方入口（如上博、南博）；
- 有些对象当前只能先补到官网定位或页脚版权信息（如本轮的陕历博）；
- 有些对象仍待继续突破（如湖南博物院）。

这进一步证明，`mils` 的比较研究不应把不同证据等级混为一谈，而应持续坚持：
1. 已核验正文；
2. 已确认官方入口但正文待补提取；
3. 已确认官网平台/页脚权利信息但未见专门规则页；
4. 待继续补证。

### 3. 国内馆的数字资源治理常与官网平台结构交织在一起
上海博物馆本轮的一个重要发现，不是条文本身，而是它把版权声明、隐私条款与“数字文物库”“三维藏品”“图录”等栏目并列放在同一站点结构中。这提示：
- 国内馆的图片使用治理，往往不是孤立存在的法律说明页；
- 而是嵌在数字资源展示、平台访问和官网内容治理结构中的一部分。

### 4. 页脚版权表达虽然简单，但仍是制度设计的重要参照
本轮上博与陕历博至少都可稳定确认页脚中的“版权所有”表达。这类表达虽然非常基础，但在制度原型设计中依然有意义，因为它提示：
- 国内馆会先通过普遍性的权利归属表述设立默认边界；
- 然后再由更细的授权、申请、交付和服务文本去承接复杂情形。

## 五、对 `mils` 的直接启发
1. 国内案例比较框架可以进一步细化为四档证据状态，而不只三档；
2. 在制度原型的“对外公开规则层”中，除版权声明、开放资源说明、申请说明页、服务办理说明外，还应继续注意“官网平台定位说明”这一更基础的外部表达；
3. 对国内博物馆而言，很多真实治理结构是“平台/网站一般规则 + 个案授权说明 + 服务办理说明”的组合，而不是一页完整制度；
4. 后续若继续补证国内案例，建议优先寻找：
   - 页脚或站内菜单中稳定出现的版权/隐私/使用规则入口；
   - 数字资源库、图录、三维文物等栏目与版权页面之间的关系；
   - 是否存在专门影像授权、联系部门、申请函下载或图片服务说明页面。

## 六、本轮阶段性结论
第二轮国内案例研究虽然没有大幅新增“已核验正文”的对象，但明显提升了国内案例的证据结构质量：
- 上海博物馆：已更稳确认官方版权/隐私入口与站点中的数字资源栏目结构；
- 南京博物院：已继续确认官方版权/隐私路径有效，但正文仍待进一步提取；
- 陕西历史博物馆：已补到官网平台性质说明和稳定的版权所有表达；
- 湖南博物院：当前仍待继续突破。

这使 `mils` 对国内博物馆规则现状的理解更进一步：国内治理往往不是“完整公开制度一次呈现”，而是由官网平台定位、版权/隐私入口、数字资源栏目和个案授权机制共同构成的分层结构。
