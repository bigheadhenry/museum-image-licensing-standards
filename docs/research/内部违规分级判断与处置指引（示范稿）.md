# 内部违规分级判断与处置指引（示范稿）

> 说明：本示范稿用于协助馆内承办、审核、审批及相关管理人员在发现图片使用不合规情形后，统一判断违规等级、选择处置工具、把握升级节奏，并保持不同个案之间的尺度基本一致。该文本为内部工作指引，不直接替代具体决定文书；具体办理时，仍应结合《整改通知单》《授权撤销或终止通知单》《风险名单记录表》《风险名单复核意见单》《后续申请强化审查意见单》《暂停受理或限制后续申请决定单》等配套文书使用。

## 一、适用目的
本指引用于回答以下内部办理问题：
1. 发现问题后，属于轻微偏差、一般违规、重点违规还是高风险违规；
2. 应优先采取提醒、补正、整改，还是直接升级到声明、撤销/终止、名单管理或后续申请限制；
3. 不同经办人面对相近情形时，如何尽量保持判断标准一致；
4. 何时应顺序升级，何时可以并行启动，何时应直接跳级处理。

## 二、基本判断原则
1. **先判断事实，再判断等级**：先确认行为事实、传播范围、合作链条、商业化程度和既往处理记录，再判断属于何种等级；
2. **先判断风险，再选择工具**：同类行为在不同传播范围和信用状态下严重程度可能不同，不宜机械按行为名称直接套用措施；
3. **轻微事项优先补正，重点事项强调控制，高风险事项强调立即止损**；
4. **整改、声明、撤销/终止、名单管理与申请限制并非互斥关系**，必要时可组合使用；
5. **保留留痕**：无论采取何种级别措施，均应形成基本记录，避免仅停留在口头沟通。

## 三、建议的四级判断框架

### 一级：轻微偏差 / 可即时补正事项
可参考判断：
- 问题主要属于格式、表述、技术性偏差；
- 尚未形成较大传播；
- 可在较短时间内更正；
- 未见明显规避管理、商业扩散或误导性表达；
- 主体配合度正常。

优先处置：
- 提醒；
- 补正、更正；
- 留痕记录。

通常不直接启动：
- 对外声明；
- 撤销/终止；
- 风险名单；
- 暂停或限制后续申请。

### 二级：一般违规 / 需正式整改事项
可参考判断：
- 已形成明确违规事实；
- 已有一定传播或实际使用；
- 仍有明确整改空间；
- 风险主要表现为条件履行不到位，而非明显恶意规避；
- 如及时整改，通常仍可恢复到可控状态。

优先处置：
- 书面整改通知；
- 限期更正、删除、补加声明、补办手续；
- 视情补缴情形费用；
- 对重复发生者，可纳入一般关注。

### 三级：重点违规 / 需升级控制事项
可参考判断：
- 重复发生一般违规；
- 出现擅自转载、再传播、转授权、平台扩散等行为；
- 已有较明显商业化、平台化或合作链条风险；
- 已收到整改要求但整改不到位、拖延整改或部分拒绝整改；
- 已明显影响馆方管理秩序、授权信赖基础或公众识别。

优先处置：
- 整改通知 + 风险名单；
- 后续申请强化审查；
- 视情同步声明；
- 视情撤销/终止；
- 对交付、平台、成果反馈和上线复核提高控制要求。

### 四级：高风险违规 / 需直接升级与并行处置事项
可参考判断：
- 冒名馆方、虚假合作、误导性背书；
- 被撤销/终止后仍继续使用；
- 未经授权商业营销、收费传播、平台售卖、二次分发；
- 虚假申报、隐瞒实际使用主体、明显规避审查；
- 已造成较大公共传播、声誉风险或管理秩序损害；
- 继续受理其新申请将显著增加风险。

优先处置：
- 严正声明 / 对外声明；
- 停止、删除、下架要求；
- 撤销/终止；
- 纳入重点关注或高度关注；
- 暂停受理或限制后续申请；
- 必要时转法务或更高层级处理。

## 四、内部判断时的五个关键维度
经办和审核时，建议至少从以下五个维度交叉判断：

### 1. 行为性质
- 是技术性偏差，还是明显超范围、规避管理、虚假申报、冒名或误导？

### 2. 传播范围
- 是局部、单次、单平台，还是多平台、持续性、大范围传播？

### 3. 商业化程度
- 是否已经涉及收费、广告营销、联名推广、品牌合作、平台经营或其他营利安排？

### 4. 配合与信用状态
- 是首次发生且配合整改，还是重复发生、拖延整改、拒不整改、规避整改？

### 5. 对馆方秩序与声誉的影响
- 是否已影响馆方授权秩序、公众识别、资源边界表达或外部声誉？

## 五、建议的处置路径选择方法

### （一）顺序升级
适用于：
- 风险从低到高逐步显现；
- 主体仍有一定配合意愿；
- 事项尚未明显公共化。

建议路径：
提醒/补正 → 整改 → 名单管理/撤销终止 → 强化审查 → 复核/解除

### （二）并行升级
适用于：
- 风险已经公开化；
- 单一措施不足以控制；
- 既需要对相对方提出要求，也需要对外部公众划定边界。

建议路径：
整改 + 声明 + 撤销/终止评估 + 名单管理 / 后续申请限制

### （三）直接跳级
适用于：
- 明显恶意；
- 明显商业化扩散；
- 明显冒名或虚假合作；
- 明显公共风险或继续放任将造成更大损害。

建议路径：
直接进入高风险处置组合，不再经过普通提醒或一般整改观察阶段。

## 六、工具选择的简明对照
1. **提醒 / 补正**：适用于轻微偏差、技术性问题、快速可修复问题；
2. **整改通知**：适用于已形成正式违规、但仍有明确整改空间事项；
3. **对外声明 / 严正声明**：适用于已进入公共传播、公众误解风险较高、冒名、商业误用或需公开划界事项；
4. **撤销 / 终止**：适用于授权基础被破坏、主体严重违反条件、整改不到位或风险已不可接受事项；
5. **风险名单记录**：适用于重复违规、严重违规、信用存疑或需持续跟踪事项；
6. **后续申请强化审查**：适用于风险尚未高到完全限制入口，但后续申请必须提高材料、条件和控制要求的事项；
7. **暂停受理 / 限制后续申请**：适用于高风险主体，或继续受理将明显增加管理风险的事项。

## 七、内部办理注意事项
1. **不要把“整改完成”直接等同于“风险已解除”**；
2. **不要把“尚可继续申请”直接等同于“可以按普通路径办理”**；
3. **不要只看申请主体表面身份，要核实实际使用主体与合作链条**；
4. **不要只看单次行为，要看是否存在重复发生、平台扩散与商业化链条**；
5. **对高风险事项，既要看当前违规，也要看继续开放申请入口是否会进一步放大风险**。

## 八、与配套文书的衔接建议
- 发现轻微偏差：以记录 + 补正为主；
- 进入一般违规：优先使用《整改通知单》；
- 进入重点违规：同步考虑《风险名单记录表》与《后续申请强化审查意见单》；
- 进入高风险违规：同步考虑《对外声明与严正声明》《授权撤销或终止通知单》《暂停受理或限制后续申请决定单》；
- 风险下降后：进入《风险名单复核意见单》所承接的复核与解除路径。

## 九、阶段性结论
本指引的核心目标，不是代替个案判断，而是为馆内提供一个相对稳定的共同判断框架：
- 先看事实；
- 再看风险；
- 再分等级；
- 再选工具；
- 再决定顺序升级、并行升级还是直接跳级。

这样可以使馆内在处理图片授权违规事项时，逐步从“经验判断”转向“分级治理 + 工具组合 + 文书承接”的制度化执行方式。
