# 图片授权申请补充材料与补正说明（示范稿）

> 说明：本示范稿用于博物馆官方网站、数字资源平台或图片授权办理页面中的补充说明文本。其目标是帮助申请人理解：什么是“补充材料”或“补正”，为什么馆方会要求补正，常见需要补充哪些内容，收到补正通知后应如何处理，以及逾期未补正可能产生什么影响。该文本不替代正式补正通知，而是用于提前公开说明补正机制，减少申请人误解与重复沟通成本。

## 一、什么是“补充材料”或“补正”？
在图片授权办理中，“补充材料”或“补正”通常是指：
- 申请已提交，但当前材料尚不足以支持馆方继续受理、审核或作出明确决定；
- 馆方要求申请人补交、补充、说明、修正或更新相关信息。

补正并不当然等于申请被拒绝，也不当然意味着申请存在严重问题。很多情况下，补正只是为了让申请事项达到可判断、可审核、可留痕的最低要求。

## 二、为什么馆方会要求补正？
馆方要求补正，通常是因为当前申请材料存在以下一种或多种情况：
1. 申请主体信息不完整；
2. 实际使用主体、发布主体、合作方关系不清；
3. 图片用途说明过于笼统，无法准确判断使用场景；
4. 使用平台、传播范围、期限、语种、地区等关键信息缺失；
5. 拟使用图片对象、数量、位置或规格说明不足；
6. 是否涉及修改、封面使用、主视觉使用、再传播、再制作等高风险行为未说明；
7. 主张学术、教育、公益或减免，但未提交相应证明材料；
8. 涉及跨境传播、商业合作、联名推广、复杂合作链条或第三方权利情形，需要进一步说明；
9. 已提交材料之间存在明显不一致、表述冲突或事实不清。

补正的核心目的，不是增加申请负担，而是避免馆方在事实不清的情况下直接作出不准确判断。

## 三、常见需要补充的材料或说明有哪些？
根据具体事项不同，馆方可能要求申请人补充以下一种或多种内容：

### （一）主体关系说明
例如：
- 申请主体与实际使用主体的关系；
- 委托办理、代理申请或合作办理关系；
- 出版社、作者、设计方、发行方、平台方之间的分工说明；
- 品牌合作、联名项目、共建项目中的主体关系说明。

### （二）项目与用途说明
例如：
- 项目名称、项目性质、项目时间安排；
- 图片拟使用的具体页面、位置、栏目、章节或传播场景；
- 是否营利、是否销售、是否投放广告、是否进入电商或平台分发；
- 是否涉及境外传播、多语言发布或多平台同步传播。

### （三）图片需求与文件规格说明
例如：
- 拟申请图片对应的资源编号、藏品名称或页面链接；
- 图片数量与具体用途对应关系；
- 是否需要高分辨率文件、出版级文件或专项服务；
- 是否需要重新扫描、重新拍摄、格式处理或加急支持。

### （四）修改与再利用说明
例如：
- 是否拟裁剪、拼接、加字、加特效、叠加设计元素；
- 是否拟用作封面、海报、主视觉、广告主图；
- 是否拟制作短视频、预告片、平台封面、社交媒体物料；
- 是否拟上传平台、素材库或允许第三方继续传播。

### （五）证明文件或支持材料
例如：
- 单位介绍信、项目立项文件、录用通知、课程计划、展览合作函；
- 合作协议摘要、项目合同关系说明、品牌联名说明；
- 样稿、版式示意图、视觉方案、传播方案、页面草图；
- 学术、教育、公益或减免申请所需的证明材料。

## 四、哪些情形更容易被要求补正？
以下情形通常更容易触发补正：
1. 申请表仅填写“研究用途”“宣传用途”“出版用途”等笼统表述，未进一步说明实际传播方式；
2. 图片对象未写明具体编号、名称或链接，馆方无法确认对应资源；
3. 申请主体与发布主体不一致，但未解释关系；
4. 明显涉及商业传播、联名合作、广告投放或跨境传播，却未提交进一步说明；
5. 要求减免，但未提交任何支持性证明；
6. 同一申请中不同材料前后表述不一致；
7. 申请事项已明显超出一般低风险场景，但提交材料仍按普通简化方式填写。

## 五、收到补正通知后，申请人应如何处理？
收到补正通知后，建议申请人重点处理以下事项：
1. 先逐项核对馆方要求补充或更正的内容；
2. 优先补齐影响判断的核心信息，如主体关系、用途范围、平台、期限、图片对象和风险事项说明；
3. 如对补正要求有疑问，应尽快与馆方沟通，而不要自行猜测；
4. 提交补正材料时，尽量保持结构清楚，与原申请相互对应；
5. 如项目方案已变化，应一并说明变化内容，而不是只补旧材料；
6. 如无法在原期限内完成补正，可及时说明原因并申请协调处理。

## 六、补正后会发生什么？
补正完成后，馆方通常会根据补正结果重新判断事项是否具备继续办理条件。可能出现的结果包括：
1. 补正完成，进入正式受理或继续审核；
2. 补正后仍需进一步说明，进入再次补充环节；
3. 补正后事实表明事项风险较高，转入强化审查；
4. 补正后发现用途、主体关系或传播结构与原申报明显不同，需调整申请路径；
5. 补正后仍不足以判断，馆方可能暂缓、限制或不予继续办理。

因此，补正并不是纯形式动作，而是影响后续路径分流的重要节点。

## 七、逾期未补正会有什么影响？
如申请人未在通知期限内完成补正，或补正后仍明显不足以支持判断，馆方通常可根据规则：
1. 暂不继续办理；
2. 视为申请人撤回申请；
3. 要求重新提交新的完整申请；
4. 对时间敏感事项，不承担因申请人延误导致的项目进度风险。

因此，如项目存在明确时间节点，建议申请人尽早准备材料，不要将补正理解为可以无限期拖延的环节。

## 八、补正是否会影响是否收费、是否批准？
补正本身不当然意味着：
- 一定收费；
- 一定不收费；
- 一定批准；
- 一定不批准。

但补正内容会直接影响馆方对以下问题的判断：
1. 申请事项属于何种使用场景；
2. 应适用何种授权类型；
3. 是否进入简易、标准或强化审查路径；
4. 是否适用收费、减免或专项服务安排；
5. 是否需要附加条件、缩小范围、暂缓或不予同意。

也就是说，补正是为了使这些判断建立在更准确的事实基础上。

## 九、申请人如何减少补正概率？
申请人可通过以下方式降低被要求补正的概率：
1. 在首次提交时尽量完整说明申请主体、实际使用主体和合作方关系；
2. 不只写“研究”“宣传”“出版”等笼统用途，而要写清传播形式和实际使用方式；
3. 尽可能准确标注图片对象、数量、用途位置和文件规格需求；
4. 如存在商业传播、联名合作、跨境传播、主视觉使用、修改再制作等情况，应主动说明；
5. 如申请减免，应同时提交支持性证明；
6. 如项目已有样稿、版式图、传播方案或页面草图，建议一并提交。
