# 对外声明与严正声明（示范稿）

> 说明：本稿用于补足博物馆图片授权与数字资源治理中的“声明/公告类规范文本”层。其功能不同于一般版权声明、网站使用规则或个案授权书，主要用于处理未经授权商业利用、冒名活动、误导性传播、虚假关联、平台误用及其他需要馆方公开划定边界、澄清事实、提示风险或保留追责权利的情形。

---

## 一、适用定位

### （一）文本性质
“对外声明”或“严正声明”是博物馆面向社会公众、媒体平台、合作机构、申请人及相关经营主体发布的边界性说明文本，用于：
1. 澄清事实；
2. 说明馆方正式发布口径；
3. 提示相关行为是否经过馆方授权；
4. 说明相关行为对馆方权益、声誉或公共秩序造成的影响；
5. 明确整改、停止、删除、下架、核实或联系要求；
6. 保留进一步追责或采取其他处置措施的权利。

### （二）与其他文本的区别
1. **不同于版权声明**：版权声明主要说明网站内容、图片资源或公开页面的一般权利边界；
2. **不同于授权书**：授权书用于个案许可与条件确认；
3. **不同于网站使用规则**：网站使用规则用于持续、一般化地说明可浏览、可下载、可否再利用等边界；
4. **不同于整改通知单**：整改通知单通常面向特定相对方、带有更明确的个案处理属性；
5. **声明/严正声明**更适合用于已经对外发生传播、存在误导或存在社会扩散影响的事项。

---

## 二、可适用情形

博物馆可在下列情形之一出现时，视需要发布对外声明或严正声明：

1. **未经授权使用馆方图片资源、展览资源或导览资源开展盈利活动**；
2. **借用、冒用、暗示使用博物馆名义进行宣传、售卖、推广、培训、讲解、直播、导览、课程或其他商业安排**；
3. **媒体平台、自媒体、第三方机构发布与馆方有关的不实信息、未核实信息或误导性信息**；
4. **对外传播中错误暗示图片、项目、产品、活动或观点已获馆方许可、认可或背书**；
5. **将馆方公开展示资源误解为可自由商用、可任意再传播、可再授权或可再包装销售**；
6. **在数字平台、内容平台、素材平台、商品平台或社交平台上传播方式已明显超出馆方允许边界**；
7. **其他需要由馆方公开澄清、划定边界、提示公众风险或维护馆方合法权益的事项**。

---

## 三、文本结构建议

对外声明或严正声明，原则上可包含以下要素：

1. **标题**：如《声明》《严正声明》《关于××事项的声明》《关于未经授权使用馆方图片资源的严正声明》；
2. **背景事实**：简要说明发生了什么、社会上出现了何种传播、馆方为何需要回应；
3. **馆方立场**：明确说明相关事项是否属于馆方正式发布、是否经过馆方授权、是否符合馆方规则；
4. **边界说明**：说明哪些行为属于未授权使用、误导性传播、冒名活动、超范围利用或其他不当情形；
5. **公众提示**：提示公众、媒体、平台、合作方应如何识别馆方正式口径或合法授权；
6. **整改要求**：视情要求相关主体停止使用、删除内容、停止售卖、下架页面、停止宣传或主动联系馆方；
7. **责任保留**：明确馆方保留进一步核实、追责、采取法律措施或其他处置措施的权利；
8. **发布信息**：落款、发布日期、必要联系方式或核实渠道。

## 四、示范文本一：关于未经授权商业利用馆方图片/导览资源的严正声明

**标题：**

# 关于未经授权使用本馆图片资源及相关公共服务资源开展盈利活动的严正声明

**正文示范：**

近日，我馆发现个别机构、个人或其他组织，未经我馆授权，擅自使用本馆图片资源、展览信息、导览内容、教育活动信息或其他与本馆相关的公共服务资源，在网络平台、社交媒体、票务平台、培训平台或其他渠道开展有偿讲解、导览、课程、推广、售卖或其他盈利性活动。

我馆在此严正声明如下：

一、我馆依法依规对馆方管理范围内的图片资源、展览资料、导览资源及相关公开信息享有相应管理权、使用边界控制权或其他合法权益。任何主体未经授权，不得擅自以我馆名义或利用我馆相关资源开展超出公开规则边界的经营性活动。

二、我馆面向公众提供的部分展示、讲解、教育或数字资源服务，系基于公共文化服务目的设置。相关资源向公众开放，并不当然意味着任何主体可以据此进行商业包装、转售、收费导览、课程售卖、再分发或其他盈利性使用。

三、任何机构、个人或其他组织，未经我馆明确授权，不得以“本馆合作”“本馆授权”“官方讲解”“官方导览”“官方指定”“馆方认可”或其他易使公众误解的表述，对外发布、宣传、售卖或组织相关活动。

四、请相关主体立即停止未经授权的使用、宣传、售卖、推广及其他相关行为，并及时删除、下架、撤回已发布内容。如已经发生交易、传播或其他社会扩散影响的，应主动纠正并妥善处理相关后果。

五、请广大公众、媒体平台、合作机构及其他相关方，以我馆官方网站、官方账号、官方公告或我馆正式书面文件作为识别馆方信息与授权状态的依据。对未经核实的信息、未经授权的合作宣传或疑似冒名活动，请审慎识别。

六、对于未经授权使用我馆资源、冒用我馆名义、误导公众、损害我馆合法权益或社会声誉的行为，我馆保留进一步核实情况、要求整改、追究责任及依法采取相应措施的权利。

特此声明。

××博物馆  
×年×月×日

## 五、示范文本二：关于不实报道或误导性传播的声明

**标题：**

# 关于有关本馆事项不实报道及误导性传播的声明

**正文示范：**

近日，部分媒体、自媒体账号、平台用户或相关机构发布、转载、传播与我馆有关的信息、通知、活动安排、资源开放情况、收费事项或合作信息。经核实，其中部分内容并非我馆官方正式发布，且存在表述不完整、事实不准确、信息失真或易引发公众误解等问题。

为避免误导社会公众，维护信息发布秩序，我馆声明如下：

一、凡涉及我馆的重要通知、资源开放安排、授权政策、收费事项、合作活动及其他正式信息，均应以我馆官方网站、官方账号、正式公告或经我馆确认的正式文件为准。

二、任何主体未经核实，不得擅自转载、扩散、加工、解读或发布可能导致公众误解的相关信息，不得以未经确认的信息对外作出宣传、售卖、引流、组织活动或其他安排。

三、对已经发布不实信息、误导性信息或未核实信息的相关主体，请立即停止传播，并及时删除、更正或澄清相关内容。

四、对因不实传播导致公众误解、交易风险、秩序影响或馆方声誉受损的，我馆保留进一步核实与采取相应措施的权利。

特此声明。

××博物馆  
×年×月×日

## 六、使用注意事项

1. 声明/严正声明应当以事实为基础，避免情绪化表达；
2. 涉及具体主体、平台、交易行为或争议事项时，应先完成必要核实并保留基础证据；
3. 如涉及较大争议、合作关系、第三方权利或潜在法律风险，宜先经法务审查或馆内授权程序后发布；
4. 对外声明宜尽量同时提供“公众识别正规渠道”的路径，如官方网站、官方账号或联系渠道；
5. 对已经进入个案处置程序的事项，声明文本可与整改通知单、撤销/终止通知单、风险名单记录等内部处置文书配合使用。

---

## 七、对制度原型的意义

将“对外声明与严正声明”纳入博物馆图片授权制度原型，意味着外部规则体系不再只包括：
- 版权声明；
- 网站使用规则；
- 开放资源说明；
- 申请说明；
- 服务办理说明；
- 下载、栏目、详情页提示；
- API 使用提示。

还应包括一类专门面向高风险传播情形的公开边界文本，用于处理：
- 未经授权商业利用；
- 冒名或误导性合作宣传；
- 不实报道；
- 平台误用与越界传播；
- 馆方权益与声誉保护。

这类文本特别符合国内博物馆实践中“分布式页面网络 + 公开声明/公告 + 个案办理路径并行”的现实结构。
