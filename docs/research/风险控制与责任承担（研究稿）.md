# 风险控制与责任承担（研究稿）

## 一、本轮目标
在已形成“授权类型与收费减免模式”“申请材料与受理要求”“审核与审批流程”“交付方式与使用要求”“署名、声明与限制条件”等研究稿的基础上，继续补齐制度中的闭环控制部分，即“风险控制与责任承担”。

本稿重点在于明确：
- 博物馆图片授权工作中究竟要控制哪些风险；
- 风险识别应嵌入申请、审核、审批、交付和后续使用的哪些环节；
- 申请人、实际使用人、合作方与馆方各自承担何种责任；
- 哪些控制措施可直接转化为制度条文与表单字段。

## 二、基本判断：图片授权风险不只是“侵权风险”
在博物馆图片授权制度中，风险控制若只理解为版权侵权防范，会导致制度过窄。围绕文物图片授权，至少还涉及以下几类风险：
1. **权利基础风险**：馆方对图片或相关资源是否具有明确管理、提供或许可基础；
2. **用途偏离风险**：申请用途与实际使用用途不一致，或后续发生超范围使用；
3. **传播失控风险**：图片在数字平台上被再次上传、下载、转载、转授权或素材化扩散；
4. **修改失真风险**：图片被过度裁剪、拼接、商业化包装或改变原有语境；
5. **声誉与公共影响风险**：文物图片被用于不当营销、误导性背书、争议议题或低俗化表达；
6. **程序与留痕风险**：审批条件不清、交付记录缺失、责任主体不明，导致后续难以追责；
7. **合作链条风险**：申请人、委托方、发布方、制作方不一致，实际控制图片的人并非获准主体。

因此，风险控制应理解为一套“前端识别—过程限制—事后追踪”的综合机制，而不是末端补救。

## 三、建议的风险分类框架

### （一）低风险事项
通常包括：
- 明确开放或低限制资源；
- 研究、教学、公益传播等低外溢场景；
- 使用范围清晰、修改需求很少；
- 申请主体、实际使用主体、发布主体一致。

低风险事项可适用简化控制，但仍应保留：
- 基本材料记录；
- 来源署名要求；
- 使用范围说明；
- 必要的事后抽查权。

### （二）中风险事项
通常包括：
- 正式出版、展览、媒体专题、平台栏目等常规但具有外部传播效果的使用；
- 需要较高规格文件；
- 涉及一定程度的设计排版、版式调整或视频编辑；
- 存在跨平台传播、较长使用期限或较大受众覆盖。

中风险事项通常应走标准路径，并配置：
- 完整审核意见；
- 明确审批条件；
- 交付规格控制；
- 成果反馈与留档要求。

### （三）高风险事项
通常包括：
- 商业广告、品牌推广、联名合作、商品营销；
- 拟将文物图片作为主视觉、品牌元素或高强度传播素材使用；
- 涉及明显改编、二次创作、IP化开发、训练素材化利用；
- 涉及敏感文物、限制公开资源或特殊协议资源；
- 申请主体与实际使用主体不一致，或存在多层合作外包关系；
- 可能引发公众误读、历史误读、文化误用或声誉争议。

高风险事项宜进入强化审查，并可增加：
- 更高层级审批；
- 法务/管理/业务联合审查；
- 额外声明、承诺或合同条款；
- 更严格的交付、样本反馈和后续监测要求。

## 四、风险控制应嵌入的关键环节

### （一）申请环节：先把风险信息收上来
申请表单不应只收“申请人是谁、想要哪张图”，还应收集能支持风险判断的信息，例如：
- 实际使用项目名称；
- 使用平台、传播渠道与发行范围；
- 是否商业用途、是否收费发行、是否带货或广告合作；
- 是否存在裁剪、拼接、配文、上字幕、加设计元素等修改；
- 是否委托第三方制作、发布或运营；
- 是否会在多个平台、多个版本、多个地区持续使用。

这些字段直接决定后续能否识别高风险事项。

### （二）审核环节：把风险判断写成书面意见
审核意见中不应只有“同意/不同意倾向”，还应写明：
- 风险点是什么；
- 风险等级如何；
- 采取何种限制条件才能降低风险；
- 是否需要补充材料、补充承诺、上提审批或加强交付控制。

### （三）审批环节：风险控制要固化为条件
审批结果如果只写“同意”，但不写明附加条件，风险控制就会落空。因此，对中高风险事项，审批结果宜固化为：
- 使用范围限制；
- 禁止修改或仅允许何种技术处理；
- 禁止再传播、再授权、素材化上传；
- 必须附特定署名与声明；
- 必须反馈样本或成品；
- 超范围使用须重新申请。

### （四）交付环节：风险控制最终要落在文件与渠道上
风险控制不能只停留在纸面，应通过交付手段落实，例如：
- 只交付与核准用途匹配的规格；
- 高风险事项不直接交付最高质量文件；
- 限定接收账号、下载期限或接收主体；
- 必要时附水印、标识或项目识别信息。

### （五）后续使用环节：应保留监督与纠偏权
制度中应明确馆方保留：
- 要求提交样本、链接或成果说明；
- 对已授权事项进行抽查；
- 对超范围使用要求纠正、删除、下架或补办授权；
- 对严重违规行为采取暂停后续申请、列入风险名单或追责措施。

## 五、责任承担的制度化结构

### （一）申请人责任
申请人至少应承担以下责任：
1. 保证申请材料真实、完整、准确；
2. 按批准范围和条件使用图片；
3. 不得擅自变更用途、平台、合作主体或传播方式；
4. 不得擅自修改、转授权、再传播或另作商业化开发；
5. 按要求署名、声明并提交成果反馈。

### （二）实际使用人/合作方责任
在实践中，申请人常常不是最终发布者。因此制度可明确：
- 申请人委托第三方制作、运营、发布的，应确保第三方遵守授权条件；
- 因第三方实际操作导致违规的，申请人仍应承担相应责任；
- 对于明确的实际使用主体，可要求其共同签署承诺或在申请中一并列明。

### （三）馆方责任
制度也应写明馆方责任边界，例如：
- 对授权事项作出审查与管理决定；
- 按批准内容提供相应文件或说明；
- 对授权条件和限制事项作出明确书面表达；
- 对明显违规行为依法依规采取处置措施。

同时也应明确，馆方提供图片并不意味着：
- 对使用成果承担内容责任；
- 对申请项目提供背书；
- 对申请人超范围行为承担责任。

## 六、违规情形的建议分类
为提高制度可执行性，建议将违规情形至少分为以下几类：

### （一）一般违规
例如：
- 未按要求完整署名；
- 未及时提交成果样本；
- 对轻微技术处理未作说明但未造成实质外溢。

可优先要求补正、更正或限期说明。

### （二）较重违规
例如：
- 超出核准平台、媒介、期限或项目范围使用；
- 擅自向第三方转交文件；
- 未经批准进行明显裁剪、拼接或再制作；
- 在公众传播中删除或规避限制性声明。

可采取：
- 停止使用；
- 删除/下架/更正；
- 暂停该主体后续申请；
- 补办授权或追缴相关费用。

### （三）严重违规
例如：
- 虚假申报；
- 将图片上传至素材库或开放下载平台；
- 擅自商业化扩散、二次授权或作为商品化资源使用；
- 将文物图片用于明显损害馆方声誉、歪曲历史文化信息或引发重大争议的场景。

可采取：
- 撤销授权；
- 永久或长期限制后续申请；
- 依法追究违约、侵权或其他责任；
- 视情形保留公开澄清、通知合作方或采取进一步法律措施的权利。

## 七、风险控制与其他章节的衔接关系
本章不应孤立存在，而应与前述章节形成对应关系：
- 与“申请材料与受理要求”衔接：通过字段设计前置风险识别；
- 与“审核与审批流程”衔接：通过分类分级形成不同审查路径；
- 与“交付方式与使用要求”衔接：通过文件规格和渠道控制降低扩散风险；
- 与“署名、声明与限制条件”衔接：通过标准提示减少误用和公众误解；
- 与“收费与减免规则”衔接：高风险和高控制成本事项可在收费中反映服务与管理成本。

## 八、可直接转化为制度条文的表达方向
“风险控制与责任承担”章节可进一步转写为以下条款结构：
1. 馆方对图片授权事项实行分类分级风险管理；
2. 对低风险、中风险、高风险事项分别适用简化、标准或强化控制措施；
3. 审核与审批应围绕权利基础、用途偏离、传播扩散、修改失真、合作链条和声誉影响等风险开展判断；
4. 申请人应对申请材料真实性、使用行为合法合规和合作方管理承担责任；
5. 对违规使用、超范围使用、擅自再传播或转授权等行为，馆方有权要求停止使用、删除下架、补办手续、撤销授权并追究责任；
6. 馆方提供图片不构成对申请项目、产品、活动或观点的认可与背书。

## 九、本轮阶段性结论
本轮研究形成六项可直接支撑制度起草的结论：
1. 博物馆图片授权风险至少应覆盖权利基础、用途偏离、传播失控、修改失真、声誉影响、程序留痕和合作链条七类；
2. 风险控制必须前置到申请字段设计，而不能只在末端追责；
3. 风险判断应写入审核意见，并转化为审批条件和交付限制；
4. 责任承担应同时覆盖申请人、实际使用人/合作方与馆方责任边界；
5. 违规情形可分为一般、较重、严重三级，以便匹配不同处置措施；
6. 风险控制是连接申请、审核、交付、使用监督与责任追究的总枢纽。
