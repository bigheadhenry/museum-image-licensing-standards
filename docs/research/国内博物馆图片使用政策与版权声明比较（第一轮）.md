# 国内博物馆图片使用政策与版权声明比较（第一轮）

## 一、本轮目标
在完成多轮国外博物馆开放获取与授权政策研究之后，本轮开始转入国内博物馆第一轮案例整理。由于国内很多馆公开披露的不是完整的“图片授权制度”，而是分散在网站中的版权声明、影像授权入口、转载要求或联系信息，因此本轮重点不是追求一次性拿到完整制度，而是：
- 优先核验官方公开入口和正文；
- 区分“版权声明型”与“授权申请型”页面；
- 从中提炼国内馆当前较常见的规则表达方式；
- 为后续制度草案中的“公开规则页/申请说明页”设计提供国内参照。

## 二、本轮已核验案例

### 1. 故宫博物院（已核验正文）
本轮核验页面：
- 影像授权
- 版权声明

可确认要点：
1. 故宫博物院明确表示，其对“拥有版权的影像资源”提供授权服务；
2. 可提供的授权用途包括论文、出版、展览、宣传及其他具有明确用途目的的使用；
3. 申请方式较为直接：下载申请函，按要求提供材料并通过指定邮箱或传真提交；
4. 页面区分了中国大陆与港澳台及海外的联系路径；
5. 其版权声明强调：网站刊载的图文、音频、视频、网站版式、网页设计等内容，除特别注明外，均为故宫博物院著作权所有；
6. 未经书面许可，不得用于商业用途；如需使用相关资料，应提前联系相关部门；
7. 转载、引用、摘编、下载等使用方式，原则上须注明作者并标明出处为“故宫博物院网站”或官网网址；
8. 未经故宫博物院或相关权利人书面许可，不得修改所使用内容；
9. 声明还明确了第三方链接不构成认可或推荐，并保留不保证声明与免责声明。

制度启发：
- 国内馆常以“版权声明 + 联系部门/授权入口”组合表达图片使用规则；
- 在公开表达上，往往先强调“未经许可不得商业使用”，再开放具体授权申请渠道；
- 来源标注与禁止擅自修改，是国内馆常见且较稳定的底线条件；
- “馆方不构成认可/推荐”的表述，在国内网站版权声明里也具有明确对应物。

### 2. 中国国家博物馆（已核验正文）
本轮核验页面：
- 版权声明

可确认要点：
1. 中国国家博物馆版权声明与故宫的结构高度接近，说明国内大型博物馆在网站版权治理上存在比较稳定的通用表达；
2. 网站刊载的图文、音频、视频、网页设计等内容，除特别注明外，均为中国国家博物馆著作权所有；
3. 任何单位或个人未经事先书面许可，不得将相关内容用于商业用途；
4. 如需使用相关资料，应提前与馆方相关部门联系；
5. 转载、引用、摘编、下载等使用时，原则上须注明作者，并标明出处为“中国国家博物馆网站”或官网网址；
6. 未经中国国家博物馆或相关权利人书面许可，不得修改所使用内容；
7. 同样保留了侵权通知、不保证声明、第三方链接不构成认可和免责声明等条款。

制度启发：
- 国内馆现有公开规则中，一个非常常见的路径是：
  “网站内容原则保护 → 商业用途须书面许可 → 非商业使用也需注明来源 → 修改受限制 → 如需进一步使用请联系馆方”；
- 这说明在国内制度设计中，公开规则页与版权声明页仍然是非常重要的制度外部接口；
- 也说明“网站版权声明”虽然不是完整授权制度，但已承担了初步分流、风险提示和外部约束功能。

## 四、本轮比较得到的国内阶段性判断

### 1. 国内馆公开表达更常以“版权声明”起步
与国外馆常见的“Open Access / Image Licensing / FAQ”并行结构不同，国内馆本轮更明显的特点是：
- 先通过网站版权声明确立一般性控制规则；
- 再通过联系方式、申请函下载、相关部门对接等方式处理个案授权。

也就是说，国内规则的公开表达更偏向：
**原则性禁止 + 个案联系授权**。

### 2. “商业使用须书面许可”是高频底线
至少从故宫和国博的公开表述看，国内馆较稳定的底线包括：
1. 未经许可不得商业使用；
2. 使用网站内容时需注明来源；
3. 未经许可不得擅自修改；
4. 如需进一步使用，应联系馆方相关部门。

这为 `mils` 制度设计中的“附条件开放型 / 受限许可型”提供了非常现实的国内参照。

### 3. 国内公开规则普遍带有强烈的网站治理属性
目前能稳定拿到的文本，多数首先是在规范“网站内容”的转载、引用、下载和修改，而不是专门为“文物图片授权管理制度”而写。

这意味着：
- 国内已有做法中，“网站版权声明”与“正式图片授权制度”之间常常存在层级差；
- `mils` 的价值之一，正是把这种分散表达进一步制度化、分类化和模板化。

### 4. 国内馆同样高度重视非背书与责任边界
虽然未必直接使用“非背书”这一术语，但故宫、国博都明确表示：
- 对第三方链接信息不作认可或推荐；
- 不对网站信息的完整性、准确性或使用后果承担广泛责任。

这说明国内制度文本同样需要保留：
- 认可/推荐边界；
- 风险提示；
- 责任边界说明。

## 五、对 `mils` 的直接启发
1. 后续制度原型包中，除了正式制度与 FAQ 外，还可考虑增设一份更接近国内馆表达习惯的《版权声明与网站使用规则（示范稿）》；
2. 在面向公众的说明体系中，应区分：
   - 网站内容一般规则；
   - 图片授权申请规则；
   - 已开放资源的直接使用规则；
3. 国内馆经验表明，“来源标注”“不得擅自修改”“商业用途须书面许可”是最稳定的基础条款，适合继续保留为标准表达层；
4. 若后续继续补国内案例，可重点关注是否存在：
   - 单独的影像授权页；
   - 申请函下载；
   - 具体收费说明；
   - 学术/公益减免逻辑；
   - 更细化的成果反馈要求。

## 六、本轮阶段性结论
本轮虽然尚未拿到很多国内馆的完整授权制度文本，但已能形成一个较清晰的国内现状判断：
- 国内博物馆公开图片规则往往不是完整地呈现为“制度条文”，而是分散体现在版权声明、联系说明和授权入口中；
- 其公开表达更偏向“原则保护 + 个案申请 + 联系部门对接”；
- `mils` 项目的制度价值，正体现在把这些零散规则系统化为“分类授权 + 标准模板 + FAQ/公开规则页”的完整治理结构。
