# 国内博物馆政策比较卡片（第二批）

> 说明：本文件用于继续按统一格式整理国内博物馆图片使用政策、版权声明与授权入口案例，并特别标记不同对象当前的证据等级。

## 1. 上海博物馆
- 机构：上海博物馆
- 已核验程度：已确认官方版权/隐私入口，正文待补提取
- 官方页面：`/mu/frontend/pg/copyright` / `/mu/frontend/pg/privacy`
- 本轮可确认结构：站内存在“隐私条款”“版权声明”官方入口，并与“数字文物库”“典藏精选”“三维藏品”“AR藏品”“图录”等栏目同处一个公开站点结构中
- 可确认权利表达：页脚可见“Copyright © 2015 ShanghaiMuseum.net 上海博物馆 版权所有”
- 当前限制：尚未稳定抽出版权声明正文，不能贸然判断其具体使用条款
- 对 `mils` 的启发：国内馆图片治理常嵌在“官网一般规则 + 数字资源栏目”的平台结构中，而非独立单页制度

## 2. 南京博物院
- 机构：南京博物院
- 已核验程度：已确认官方版权/隐私入口，正文待补提取
- 官方页面：`/zh-CN/copyright` / `/zh-CN/privacy`
- 本轮可确认结构：版权与隐私路径可稳定访问，属于官方站点框架一部分
- 当前限制：页面采用前端框架渲染，当前未抽出可直接引用的正文
- 对 `mils` 的启发：国内案例研究应接受“入口已确认但正文待提取”的中间状态，避免过度归纳

## 3. 陕西历史博物馆
- 机构：陕西历史博物馆
- 已核验程度：已确认官网平台定位说明与页脚权利表达，未定位到专门图片规则页
- 官方页面：官网说明页 `/about/detail/209.html`
- 本轮可确认结构：页面明确将官网表述为与公众交流互动的重要平台
- 可确认权利表达：页脚可见“Copyright 2021 陕西历史博物馆版权所有 All Rights Reserved.”
- 当前限制：尚未定位到单独版权声明、图片授权或图片使用规则正文
- 对 `mils` 的启发：除版权声明和授权页外，官网平台定位说明本身也可能是制度外部接口的一部分

## 4. 湖南博物院
- 机构：湖南博物院
- 已核验程度：待继续补证
- 当前状态：本轮直连官网及常见版权/隐私路径时抓取出现 SSL EOF 类错误，尚未获得稳定正文或路径证据
- 对 `mils` 的启发：国内案例补证需准备多种抓取/定位路径，单一方法可能不足

## 本批卡片的共同观察
1. 国内案例的“证据等级差异”本身值得被显式记录；
2. 站点结构、菜单入口、页脚版权表达虽然不等于完整制度条文，但仍能反映公开治理框架；
3. 国内博物馆的图片治理常常是平台型、分层型表达，而不是一页完整授权政策；
4. 这进一步支持 `mils` 继续采用“网站规则 + 开放资源说明 + 申请说明页 + 服务办理说明 + 高规格服务说明”的分层制度原型方法。
