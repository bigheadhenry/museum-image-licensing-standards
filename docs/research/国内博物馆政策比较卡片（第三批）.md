# 国内博物馆政策比较卡片（第三批）

> 说明：本文件用于继续按统一格式整理国内博物馆图片使用政策、版权声明、声明公告与资源栏目入口案例，并显式标记不同对象当前的证据等级。

## 1. 湖南博物院
- 机构：湖南博物院
- 已核验程度：已核验官方声明正文，尚未核验专门图片规则页
- 官方页面：`/zh-hans/aboutus/声明`、`/zh-hans`
- 本轮可确认结构：官网首页稳定可达，具有线上展览、精品文物等数字展示属性；《声明》页可稳定访问并提取正文
- 可确认规则表达：强调对外信息应以官网正式发布为准；相关媒体及机构不得转载、扩散未经核实或不实内容；涉及采访报道应先与馆方核实
- 当前限制：本轮拿到的是声明正文，而非专门版权/图片使用规则页
- 对 `mils` 的启发：国内馆的规范性文本不一定叫“版权声明”，声明/公告也可能承担传播边界与权威来源控制功能

## 2. 苏州博物馆
- 机构：苏州博物馆
- 已核验程度：已确认官网与馆藏栏目官方入口，未见专门规则页正文
- 官方页面：`/`、`/Collection/Index`、`/Wap/Home/`、`/Wap/Collection/Index`
- 本轮可确认结构：主站、馆藏栏目页和移动端对应栏目均稳定可达，说明其官网具有面向公众的馆藏/资源访问结构
- 当前限制：受响应头兼容和提取条件影响，尚未抽出可直接引用的版权声明、图片使用规则或申请说明正文
- 对 `mils` 的启发：国内馆的公开治理接口往往先体现在资源栏目结构，而不是先给出一页集中式政策文本

## 本批卡片的共同观察
1. 国内馆的规范性表达，不一定只出现为“版权声明”，也可能表现为声明、公告、官网正式发布说明；
2. 官网首页、馆藏栏目和移动端资源入口本身，就是研究数字资源治理结构的重要证据；
3. 国内案例研究应继续坚持分层证据方法，而不是把“官网存在资源栏目”误写为“已公开完整政策”；
4. 这进一步支持 `mils` 当前采用的多层制度原型：平台层说明、栏目层提示、详情页标签、下载提示、接口提示与办理说明并行。
