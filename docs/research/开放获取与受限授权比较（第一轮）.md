# 开放获取与受限授权比较（第一轮）

## 一、本轮目标
本轮研究聚焦于博物馆图片授权制度中的一个关键分野：
- 一类机构对部分馆藏图片实行开放获取（Open Access / Open Data），允许免费、甚至商业性再利用；
- 另一类机构则保留更强的许可控制，通过权限、版权状态、来源标注、申请程序等方式进行管理。

本轮先基于可直接获取的官方公开页面，形成第一轮比较笔记，为后续制度草案中的“授权类型与基本条件”“收费与减免规则”“限制条件”章节提供支撑。

## 二、已核验的官方公开信息

### 1. 大都会艺术博物馆（The Metropolitan Museum of Art）
官方页面：Image and Data Resources

可确认要点：
- 将馆藏 2D/3D 图像区分为两类：
  1) 馆方认为属于公有领域或馆方放弃其可能拥有版权的图像；
  2) 馆方已知仍受版权或其他限制约束的图像。
- 自 2017 年起实施 Open Access 政策。
- 对其认为属于公有领域的作品图像，采用 CC0，允许不受限制使用，且不收费。
- 对整个在线馆藏的基础数据，也按 CC0 开放。

初步观察：
- 其制度逻辑不是“所有图片统一开放”，而是先按权利状态分层，再决定开放程度。
- 这说明博物馆图片授权制度中，“权利状态识别”是前置条件，而不是后置说明。

### 2. Rijksmuseum
官方页面：
- Image requests
- Information and Data Policy

可确认要点：
- 自 2011/12 起推进开放数据政策，并在 2024 年扩展为更完整的信息与数据政策。
- 明确提出：其大量馆藏信息与数据可自由使用，包括商业用途，且无需事先申请许可。
- 页面列明了不同权利状态：
  - Public Domain 或 CC0：可自由使用；
  - CC BY 4.0：可使用，但需注明 Rijksmuseum 为来源；
  - 对仍受版权保护的对象，则会在版权说明中明确限制。
- 另有图片申请页面强调：馆方持续更新高质量图片，建议使用最新版本，而不要反复沿用旧图。

初步观察：
- Rijksmuseum 的制度更接近“分层开放”：并非只有开放/不开放二元结构，而是至少包括 CC0、CC BY、受限对象三种层次。
- 该模式对制度建设的启发是：授权制度可将“来源署名要求”与“是否收费/是否需审批”拆分处理，而不必捆绑成同一控制手段。
- 其对“最新版高质量图像”的强调，也提示制度中应纳入“交付版本控制”要求。

### 3. British Museum
官方公开页面抓取受限（403），但搜索结果可确认存在与 copyright / permissions 相关的官方页面入口。

当前阶段可记录为：
- British Museum 至少存在版权与许可说明页面；
- 但在本轮环境下未直接抓到正文，后续需继续补充。

### 4. Smithsonian / Getty

## 三、第一轮比较结论

### 1. “开放获取”并不意味着取消全部制度控制
从 The Met 与 Rijksmuseum 可见，开放获取通常仍以以下前提为基础：
- 馆方已识别作品或图像的权利状态；
- 仅对满足特定条件的对象实行 CC0 或类似开放策略；
- 对其他仍受版权约束或受第三方协议限制的对象，仍保留限制或单独说明。

因此，制度草案中不能简单写成“免费即开放、收费即受控”，而应区分：
- 权利状态；
- 使用场景；
- 署名义务；
- 是否需要申请；
- 是否需要支付费用；
- 是否允许修改与再传播。

### 2. 可初步抽象出三类授权/开放模式
基于当前材料，可先提出一个研究性分类：

#### （1）开放使用型
- 适用对象：公有领域作品图像、馆方明确放弃相关权利主张的图像；
- 典型规则：CC0 / 公有领域声明；
- 主要特征：免费、可广泛再利用，通常无需个案审批。

#### （2）开放但附条件型
- 适用对象：允许使用但要求来源标注，或要求遵守一定说明义务的图像/数据；
- 典型规则：CC BY 等；
- 主要特征：可使用，但需署名或满足特定条件。

#### （3）受限许可型
- 适用对象：仍受版权、合同、隐私、文物安全、第三方权利等限制的图像；
- 主要特征：需申请、需审核，可能收费，并可能限制用途、期限、媒介和修改。

这一三分法适合进入后续制度草案中的“授权类型与基本条件”章节。

### 3. 制度设计中应单列“交付版本与质量控制”
Rijksmuseum 的做法提示：
- 图片授权不只是“准不准用”的问题；
- 还包括“交付哪个版本”“交付精度和质量标准是什么”“是否要求使用最新版文件”等问题。

因此制度草案可在“交付方式与使用要求”章节中单列：
- 文件格式；
- 分辨率等级；
- 是否提供水印版/出版版/长期保存版；
- 是否要求使用授权期内提供的最新版文件；
- 是否允许申请人自行压缩、裁剪、再加工。

## 四、对制度草案的直接启发

### 1. 可新增的规范要点
建议在制度草案后续版本中明确以下要点：
- 以“权利状态识别”为授权前置环节；
- 区分开放使用、附条件开放、受限许可三类基本模式；
- 将“来源署名要求”作为独立规则项，而不是笼统并入限制条件；
- 将“交付版本与质量控制”写入专门条款；
- 对受限许可型图片，进一步细化申请材料、审批流程、收费标准与责任边界。

### 2. 对后续研究的要求
下一轮应继续补充：
- Smithsonian、Getty、British Museum 的正式公开政策文本；
- 国内博物馆是否存在公开的图片授权申请页面、图片使用规则、收费说明；
- 国内外在“学术使用免费 / 商业使用收费”上的实际区分规则；
- 各机构对署名格式、禁止性用途、修改限制、再传播限制的写法。

## 五、本轮结论（简要）
本轮已初步证明：
- 博物馆图片授权制度的核心，不是简单的“收费/免费”二分，而是“权利状态 + 使用场景 + 条件约束 + 交付控制”的组合治理结构；
- 开放获取政策本身也依赖权利状态识别与分层规则，不等于完全放弃制度管理；
- 制度草案下一步可优先补写“授权类型与基本条件”及“交付方式与使用要求”两部分。
