# 补正通知单（示范稿）

> 说明：本示范稿用于在图片授权申请进入受理阶段后，发现申请材料不齐全、表述不清、关键信息缺失或事实之间存在明显不一致时，向申请主体发出正式补正通知。该文书与《图片授权申请补充材料与补正说明（示范稿）》《图片使用申请表（示范稿）》《审核意见单（示范稿）》《授权申请受理、审核与审批工作流程说明（示范稿）》等文本配合使用，用于把补正机制从对外说明层与流程层进一步落到正式执行文书层。

## 二、补正事由
经对贵方提交的图片授权申请材料进行初步审查，现发现申请材料存在以下一种或多种情况，暂不足以支持继续受理、审核或作出明确决定：
- 申请主体信息不完整或证明材料不足；
- 实际使用主体、发布主体、合作方关系不清；
- 项目名称、用途说明或使用场景表述过于笼统；
- 使用平台、传播范围、期限、语种、地域等关键信息缺失；
- 拟申请图片对象、数量、用途位置或文件规格说明不足；
- 是否涉及修改、封面/主视觉使用、平台分发、再传播或再制作等高风险事项未说明；
- 减免申请理由已提出，但未提交支持性证明材料；
- 已提交材料之间存在明显不一致、表述冲突或事实不清；
- 其他：

## 三、需补正事项
请贵方根据实际情况，于规定期限内补充、说明、更正或更新以下材料/信息（可逐项勾选填写）：

- 补充申请主体身份证明 / 单位证明；
- 补充统一社会信用代码、身份证明或授权代表信息；
- 补充联系人信息；
- 补充代理申请或委托关系材料；
- 说明申请主体、实际使用主体、发布主体之间关系；
- 其他：

### （二）项目与用途补正
- 补充项目名称、项目性质或项目时间安排；
- 进一步说明使用目的与使用场景；
- 说明成果形式（图书、图录、展览、视频、网站、公众号、社交媒体、广告物料等）；
- 说明是否存在发行、销售、招商、赞助、联名、广告投放或平台分发安排；
- 说明使用媒介、传播渠道、传播范围、使用期限、地域或语种；
- 其他：

### （三）图片对象与文件需求补正
- 补充拟申请图片对应的藏品名称、图片名称、资源编号、页面链接或清单附件；
- 说明各图片与具体用途位置的对应关系；
- 说明拟申请数量；
- 说明所需文件规格与交付方式；
- 说明是否需要高分辨率文件、出版级文件、重新扫描/拍摄或专项技术服务；
- 其他：

### （四）修改、再利用与传播安排补正
- 说明是否拟裁剪、拼接、局部放大、叠加文字、动画、特效或包装设计；
- 说明是否拟作为封面、海报、主视觉、广告主图或招商宣传使用；
- 说明是否拟上传第三方平台、素材库或允许第三方继续传播；
- 说明是否涉及批量下载、批量导出、二次分发或平台聚合展示；
- 说明是否存在跨境传播、多语言发布或复杂合作链条；
- 其他：

### （五）证明文件补正
- 补充立项文件、出版计划、课程计划、展览合作函、录用通知等项目证明；

- 补充样稿、版式示意图、传播方案、页面草图、视觉方案或脚本摘要；
- 补充合作协议摘要、品牌合作说明、平台合作说明；
- 其他：

## 四、补正方式与提交要求
1. 请在本通知发出之日起 **____个工作日内 / 截至____年__月__日** 完成补正；
2. 提交方式：
   - 线上表单补充提交；
   - 指定邮箱发送；
   - 系统上传补充材料；
   - 纸质材料补交；
   - 其他：
3. 请尽量按照“本通知列明事项—对应补正材料”方式逐项提交，避免仅重复提交原材料；
4. 如项目情况较原申请已发生变化，请一并说明变化内容，而不仅补交原缺失材料；
5. 如对补正要求存在疑问，请在期限届满前及时联系承办部门沟通。

## 五、补正期间的处理说明
1. 在补正完成前，本申请原则上**暂不进入下一审核/审批环节**；
2. 补正材料提交后，馆方将根据补正情况重新判断是否具备继续受理、继续审核或进入其他路径的条件；
3. 补正完成并不当然意味着申请将被批准，馆方仍将根据权利基础、使用场景、授权类型、收费/减免规则和风险控制要求继续审查；
4. 如补正后事项显示为高风险、事实重大变化或超出原申报边界，馆方可调整办理路径，转入强化审查、附条件审查、暂缓决定或其他处理方式。

## 六、逾期或补正不足的处理
如贵方存在以下情形之一，馆方可视情况作出相应处理：
1. 未在规定期限内完成补正；
2. 补正材料仍明显不足以支持判断；
3. 补正后发现核心事实前后不一致且未作合理说明；
4. 补正后事项与原申请相比已发生重大变化，需重新申请或重新提交完整材料；
5. 其他不具备继续办理条件的情形。

对应处理方式可包括：
- 暂不继续办理；
- 视为申请人撤回申请；
- 要求重新提交新的完整申请；
- 转入其他审查或限制路径。

## 九、使用说明（馆内）
1. 本通知单适用于“材料不足、表述不清、需补充说明”的正式告知，不适用于已经进入明确不同意决定的事项；
2. 应尽量一次性列明需要补正的核心事项，减少反复补正；
3. 补正通知应与申请编号、后续审核意见、审批结果和留档材料对应保存；
4. 如事项已明显超出普通补正范畴，而涉及高风险、虚假申报或严重边界问题，可不单独以补正通知替代后续限制、暂缓或不予同意处理。
