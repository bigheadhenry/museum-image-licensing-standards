# 制度草案框架

## 一、草案定位
本草案拟作为博物馆图片授权工作的基础规范文本，用于统一授权对象、适用范围、审批要求、授权条件、收费与减免、风险控制、交付方式和责任承担等方面的制度表达。

草案研究对象以博物馆管理的图片资源为范围，重点聚焦文物图片在出版、展览、传播、研究、教育和商业合作等使用场景中的授权管理。

## 五、各章可采用的核心写法

### 1. 授权类型与基本条件
可按以下四类结构起草：
- 开放使用型；
- 附条件开放型；
- 受限许可型；
- 禁止/暂不授权型。

并明确授权判断顺序应为：
“权利/管理基础识别 → 使用场景判断 → 授权类型确定 → 收费/减免适用”。

### 2. 申请材料与受理要求
本章可围绕四项判断组织条文：
- 申请主体是否明确；
- 使用事项是否清楚；
- 授权条件是否可判断；
- 是否足以支持收费与风险识别。

材料要求可至少覆盖：
- 主体信息；
- 项目与用途说明；
- 图片需求信息；
- 修改与再利用说明；
- 减免/特殊情形证明；
- 承诺与签署。

### 3. 审核与审批流程
本章宜采用“申请—受理—补正—审核—审批—收费/文书—交付—留档”的链条，并设置：
- 简易路径；
- 标准路径；
- 强化审查路径。

审批结果可至少分为：
- 同意；
- 附条件同意；
- 暂缓决定；
- 不同意。

### 4. 交付方式与使用要求
建议按文件规格分级：
- 网络传播级；
- 研究参考级；
- 出版使用级；
- 特殊制作级。

并明确：
- 核准用途、媒介、期限、平台范围；
- 允许与禁止的修改行为；
- 再传播、转授权和成果反馈要求；
- 交付不改变权利状态。

### 5. 署名、声明与限制条件
本章可分为三组规则：
- 署名规则；
- 声明规则；
- 限制条件规则。

应重点覆盖：
- 来源标注；
- 权利状态说明；
- 非背书声明；
- 转载、再传播、改编和商业外溢限制。

### 6. 风险控制与责任承担
建议引入：
- 低风险/中风险/高风险三级分类；
- 权利基础、用途偏离、传播失控、修改失真、声誉影响、程序留痕、合作链条等七类风险；
- 申请人、实际使用人/合作方、馆方责任边界；
- 一般/较重/严重三级违规及对应处置措施。

### 7. 附录模板与标准表达
建议后续同步形成至少五组配套模板：
- 申请类模板；
- 审核审批类模板；
- 授权交付类模板；
- 使用反馈与监督类模板；
- 标准署名/声明/限制提示语模板。
