# 研究范围与边界

## 一、项目定位
本项目聚焦博物馆图片授权问题的研究与规范建设，目标是在梳理国内外相关实践、政策与工作模式的基础上，形成适用于馆内管理与业务执行的图片授权规范与标准草案。

## 二、研究对象
本项目讨论的“图片授权”，是指博物馆对其管理范围内图片资源在特定使用场景下授予使用权的管理活动。这里的“使用”主要包括但不限于：
- 出版；
- 展览；
- 视频制作；
- 论文写作；
- 宣传传播；
- 教育活动；
- 学术研究；
- 媒体发布。

## 三、资源范围
本项目以博物馆管理范围内的图片资源为总体范围，以文物图片为重点研究对象，同时兼顾其他可能纳入授权管理的图片类型，如：
- 展览相关图片；
- 活动记录图片；
- 建筑与空间环境图片；
- 教育传播图片；
- 研究与业务工作过程图片。

## 四、不纳入本轮核心讨论的内容
为避免研究边界过大，当前阶段不将以下议题作为核心：
- 图片采集系统、资产管理系统或 DAM 系统的技术实现；
- 视频、音频、三维数据等非图片资源的授权规范；
- 版权法理论的全面综述；
- 一切与 MDAMS 相关的对象模型、流程设计或系统边界。

## 五、当前研究重点
当前优先研究以下问题：
1. 博物馆图片授权的常见使用场景与授权类型；
2. 国内外博物馆在申请流程、收费模式、权利声明、限制条件上的做法；
3. 文物图片授权与其他馆管图片资源授权的共性与差异；
4. 可供馆内落地的制度结构、流程框架和文本模板。
