# 审核与审批流程（研究稿）

## 一、本轮目标
在已形成“使用场景分类”“授权类型与收费减免模式”“申请材料与受理要求”等研究稿的基础上，继续向制度落地推进，补出博物馆图片授权工作中的“审核与审批流程”框架。

本稿重点不在画复杂流程图，而在明确：
- 审核与审批各自解决什么问题；
- 不同情形应进入何种审查路径；
- 如何把授权判断、收费判断、风险控制和交付前置到一个连贯流程中。

## 二、基本判断：审核与审批不是同一件事
制度设计中，审核与审批经常被混用，导致职责不清、节点重复或责任落空。更合理的区分应是：

1. **审核**：对申请事项进行事实核对、条件判断和风险识别；
2. **审批**：在审核意见基础上作出是否同意、附条件同意或不同意的管理决定。

因此，审核更偏向专业判断与材料核验，审批更偏向制度授权下的正式决定。

## 三、建议的流程分层结构

### （一）标准流程建议
可将图片授权工作划分为以下七个连续环节：
1. 申请提交；
2. 形式受理；
3. 材料补正（如需）；
4. 实质审核；
5. 审批决定；
6. 费用确认与授权文书生成；
7. 文件交付与结果留档。

这一结构的好处是：
- 申请是否合格先在前端被拦截；
- 使用目的、授权条件、收费规则在审核环节统一判断；
- 审批只对已形成审核意见的事项作决定；
- 批准后再进入收费、出证、交付，避免重复劳动。

### （二）审核环节的核心任务
审核至少应覆盖以下五项判断：
1. **授权基础判断**：馆方对相关图片是否具备管理和许可基础；
2. **用途判断**：申请事项属于何种使用场景，是否属于允许范围；
3. **类型判断**：适用开放使用型、附条件开放型、受限许可型还是禁止/暂不授权型；
4. **收费判断**：是否收费、是否减免、收费层级如何；
5. **风险判断**：是否涉及商业广告、敏感资源、再制作、高争议传播或其他复杂情形。

### （三）审批环节的核心输出
审批不宜只有“同意/不同意”两种结果，建议至少设置：
- 同意；
- 附条件同意；
- 暂缓决定（需补充说明或上会研究）；
- 不同意。

其中“附条件同意”是制度落地中的关键形式，可用于承载：
- 使用期限限制；
- 使用媒介限制；
- 传播范围限制；
- 不得修改或仅允许轻微技术性处理；
- 必须署名、注明来源、附加声明；
- 不得再许可、转授权或上传素材库。

## 四、建议的审查路径分类

### （一）简易路径
适用于以下情形：
- 已明确开放或低风险资源；
- 学术研究、教学、公益传播等低争议用途；
- 使用范围清晰、申请材料完整；
- 不涉及商业合作、明显改动或再传播安排。

简易路径可采用较简化的审核与审批程序，但仍应保留书面记录。

### （二）标准路径
适用于一般出版、展览、宣传使用等常规申请。其特点是：
- 需要完成完整审核；
- 形成明确审批意见；
- 如涉及收费，应在审批通过后进入费用确认与文件交付。

### （三）强化审查路径
适用于以下高风险或复杂情形：
- 商业广告、品牌推广、联名合作；
- 大规模复制、发行或多平台商业传播；
- 涉及敏感文物、限制公开资源或特殊协议资源；
- 涉及明显改编、再制作、拼接包装、主视觉使用；
- 申请主体、实际使用主体、发布主体不一致；
- 拟申请事项可能引发声誉、法律或社会影响风险。

强化审查路径可考虑增加：
- 更高层级复核；
- 业务、法务或管理层联合会审；
- 对外合作协议或补充承诺文件；
- 更严格的交付和使用限制。

## 五、审核意见应包含的最低内容
为避免审批流于形式，审核意见至少应覆盖：
- 申请主体及用途概述；
- 涉及图片资源范围；
- 使用场景归类；
- 拟适用授权类型；
- 是否收费及减免理由；
- 是否存在修改、传播、转授权等限制事项；
- 风险判断结论；
- 建议审批结果及附加条件。

这样可使审批决定建立在可追踪的书面基础之上。

## 六、审批决定与后续环节的衔接

### （一）与收费环节衔接
审批通过后，如属于收费事项，应根据审核意见进入：
- 费用核定；
- 缴费确认；
- 减免确认；
- 必要的协议或授权书签署。

制度上应避免“未形成批准意见先收费”或“已收费但授权条件不清”的情况。

### （二）与授权文书衔接
审批结果应尽可能转化为明确的授权文书或书面通知，载明：
- 授权对象；
- 授权用途；
- 授权范围；
- 使用期限；
- 文件规格；
- 署名与声明要求；
- 禁止事项；
- 违约后果。

### （三）与交付环节衔接
只有在审批结果明确、收费条件满足、必要文书完成后，才进入图片文件交付环节。这样可避免授权条件未固化即发生图片外流。

### （四）与结果留档衔接
审核意见、审批结果、收费记录、交付记录和申请材料应形成完整留档，以便后续：
- 核验实际使用是否合规；
- 处理争议或侵权问题；
- 统计授权数量、用途结构和收费情况；
- 反向优化制度规则。

## 七、制度写作中的几个关键控制点

### （一）应明确时限，但保留复杂事项弹性
制度可设置一般事项处理时限，但对于：
- 材料补正；
- 高风险事项；
- 联合审查事项；
- 需外部确认权利状态的事项；
可保留适当延长机制。

### （二）应避免“审批层级越多越安全”的误区
审批层级过多会导致效率低、责任分散。更合理的方式是：
- 低风险事项走简易或标准路径；
- 高风险事项才进入强化审查；
- 通过分类分级控制风险，而不是一律上提。

### （三）应保留不予批准和撤销机制
制度中应明确：
- 不符合授权条件的，可不予批准；
- 发现虚假申报、超范围使用或违反附条件要求的，可暂停、撤销或终止授权；
- 已交付图片并不意味着永久、无限制使用。

## 八、可直接转化为制度条文的表达方向
“审核与审批流程”章节后续可转写为以下条款结构：
1. 申请经受理后进入审核程序；
2. 审核应围绕授权基础、使用用途、授权类型、收费适用和风险事项进行；
3. 根据事项复杂程度，可适用简易、标准或强化审查路径；
4. 审批结果可分为同意、附条件同意、暂缓决定和不同意；
5. 对经批准的事项，应根据审批结果办理收费、授权文书和文件交付；
6. 对不予批准、补正逾期、虚假申报或违规使用的，应明确相应处理方式；
7. 授权办理全过程应留存书面记录。

## 九、本轮阶段性结论
本轮研究形成四项可直接支撑制度起草的结论：
1. 审核解决“是否符合条件、风险如何、适用何种规则”，审批解决“是否准许以及在何种条件下准许”；
2. 博物馆图片授权流程宜采用“申请—受理—补正—审核—审批—收费/文书—交付—留档”的链条；
3. 制度中应引入简易、标准、强化三类审查路径，以适应不同风险等级的事项；
4. 审批结果必须能转化为可执行的条件约束，否则后续收费、交付和责任追究将失去依据。

## 十、下一轮建议
下一轮可优先选择以下两条路径之一：
- 路径 A：继续补写“交付方式与使用要求（研究稿）”，打通从审批到实际交付的最后一段；
- 路径 B：开始把现有研究成果转写为《制度草案》中的“申请材料与受理要求”“审核与审批流程”章节初稿。
