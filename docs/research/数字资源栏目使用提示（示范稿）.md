# 数字资源栏目使用提示（示范稿）

> 说明：本示范稿用于博物馆官方网站、数字资源平台、图录页面、资源详情页或下载页中，以“页面提示语/栏目使用提示”的方式，向用户快速说明该栏目资源的基本访问边界、分流逻辑和风险提示。其定位不同于《官网平台与数字资源公开使用边界说明》：后者偏整体平台说明，本提示则更接近栏目级、页面级和详情页级的实际表达。正式实施时，可根据栏目结构、页面布局、资源类型和权利标签体系进行精简或拆分使用。

## 一、栏目提示目的
本栏目用于展示本馆数字资源、资源信息、图片版本或相关说明，方便公众浏览、检索、学习和研究参考。请注意：栏目中的资源公开展示，不当然意味着相关图片或文件可被任意复制、转载、商业使用、批量下载或再次授权。

## 二、请先判断资源类型
使用本栏目资源前，建议先判断页面所示资源属于以下哪类情形：
1. 已明确标注为开放资源，可按页面规则直接使用；
2. 仅供浏览、学习、研究参考，不当然开放下载或再利用；
3. 需联系本馆或提交申请后方可使用；
4. 需另行办理高规格文件、技术处理或专项服务。

如页面未明确标注开放规则，原则上不建议仅依据“页面可见”推定其可自由使用。

## 三、浏览、检索与下载的区别
1. **可浏览**：通常意味着可在页面内查看资源内容或相关信息；
2. **可检索**：通常意味着可查找资源名称、编号、类别或说明信息；
3. **可下载**：仅表示本馆在当前页面提供特定版本文件的获取方式，并不当然意味着该文件可被任意传播、商业使用或再次分发。

用户不应将“可浏览”“可检索”或“有下载按钮”直接理解为“可无限制使用”。

## 四、来源标注与声明要求
除页面另有明确豁免外，使用本栏目中可使用的资源时，原则上仍应保留必要来源信息，并按页面要求或本馆统一规则附署名、来源说明、限制提示语或非背书声明。

## 五、哪些情况通常需要进一步申请？
存在以下情形之一的，建议联系本馆或进入正式申请流程：
1. 拟用于图书、期刊、图录、展览、视频节目、宣传传播、广告营销、品牌合作等正式项目；
2. 拟获取高分辨率、出版级或印刷级文件；
3. 拟对图片进行裁剪、拼接、改色、包装设计或其他再制作；
4. 拟进行跨平台、大范围、持续性或跨境传播；
5. 页面规则或资源标签已明确提示需申请办理。

## 六、哪些情况通常需要另行办理服务？
即使相关资源可以在页面中浏览或按规则使用，存在以下情形之一的，仍可能需要另行办理服务事项：
1. 需要更高规格文件；
2. 需要批量打包、格式转换或技术处理；
3. 需要加急交付；
4. 需要纸质证明、补充说明或专项支持。

此类事项可结合《图片服务办理说明》《高规格文件与专项服务说明》理解和办理。

## 七、禁止误导公众
任何单位或个人不得因资源来自本馆官网、数字资源栏目、资源详情页或下载页，而使公众误认为本馆对其项目、产品、活动、传播内容或合作安排作出认可、推荐、联名或背书。

## 八、第三方权利与外部风险提示
本栏目展示的资源，不当然排除可能存在的第三方版权、商标权、肖像权、隐私权、合同限制或其他外部权利风险。用户应结合具体用途自行审慎判断并承担相应责任。

## 九、违规使用处理
对删除来源信息、超越页面规则使用、擅自商业化利用、擅自再传播、误导公众或其他违反本馆规则的行为，本馆可要求停止使用、删除下架、更正说明、补办手续、限制后续服务获取，并依法追究相应责任。

## 十、与其他规则的关系
本提示为栏目级、页面级说明文本，应与下列规则配套理解：
1. 《版权声明与网站使用规则》；
2. 《官网平台与数字资源公开使用边界说明》；
3. 《开放资源使用说明》；
4. 《图片授权申请说明页》；
5. 《图片服务办理说明》；
6. 《高规格文件与专项服务说明》。

如具体页面标签、资源详情说明、授权书或专项规则另有更具体要求的，以更具体的规则为准。
