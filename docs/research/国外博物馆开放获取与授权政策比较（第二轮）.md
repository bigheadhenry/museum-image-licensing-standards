# 国外博物馆开放获取与授权政策比较（第二轮）

## 一、本轮目标
在《开放获取与受限授权比较（第一轮）》的基础上，本轮继续围绕国外博物馆公开图片政策进行第二轮比较整理，重点不是追求一次性拿到所有机构完整文本，而是：
- 优先核验可稳定访问的官方页面；
- 对受访问限制但可确认存在的政策入口进行分类记录；
- 将比较重点进一步从“是否开放”推进到“开放对象、条件表达、受限对象和制度启发”。

## 二、本轮已核验或可确认的机构情况

### 1. The Metropolitan Museum of Art（已核验）
官方页面：Image and Data Resources

本轮复核可确认要点：
1. 馆方将馆藏作品图像区分为两类：
   - 馆方认为属于公有领域的作品图像，或馆方放弃其可能拥有版权的图像；
   - 馆方已知仍受版权或其他限制约束的图像。
2. 自 2017 年起实施 Open Access 政策；
3. 对其认为属于公有领域的作品图像，按 CC0 开放，可广泛且免费使用；
4. 对整个在线馆藏的基础数据，也按 CC0 开放。

制度启发：
- “是否开放”之前必须先做权利状态识别；
- 可将“图像开放”和“基础数据开放”分层处理；
- 不是所有馆藏图片一律适用同一开放规则。

## 三、第二轮比较结论

### 1. 国外机构常见的不是“单一规则”，而是“并行政策结构”
从目前可核验和可确认的页面结构看，国外博物馆往往并不是用一个页面解决全部问题，而是采用以下并行结构：
1. Open Access / Open Data 页面；
2. Image Licensing / Permissions 页面；
3. FAQ 或使用说明页面；
4. Terms / Rights / Conditions 页面。

这意味着制度设计时，不能只写“授权制度正文”，还要考虑：
- 对外公开规则页；
- 申请说明页；
- FAQ/标准表达页；
- 内部执行模板页。

### 2. 开放获取与受限授权往往是并存关系，而不是二选一关系
本轮材料进一步支持一个判断：
- 开放获取通常适用于已识别为公有领域或可开放的对象；
- 受限许可通常适用于仍受版权、合同、合作协议、敏感属性或传播风险约束的对象；
- 同一家机构内部，二者常同时存在。

这对 `mils` 制度设计的直接意义是：
- 制度不宜写成“统一申请—统一收费—统一授权”的单一路径；
- 更适合写成“开放使用型 / 附条件开放型 / 受限许可型 / 暂不授权型”的分层治理结构。

### 3. FAQ 与说明文本本身也是制度组成部分
Smithsonian 和其他机构的线索都提示，国外开放政策通常不仅仅是一句“可用/不可用”，而是会尽量通过 FAQ、使用说明或学习资源告诉用户：
- 哪些资源可用；
- 如何理解开放；
- 何时仍需注意来源、条件和边界；
- 如何在教育、研究和传播场景中正确使用。

这说明：
- `mils` 项目中此前形成的《标准署名与声明示例（示范稿）》并不是附属材料，而是制度的重要外部接口；
- 后续还可进一步形成“常见问题说明”或“申请人使用指南”。

### 4. 案例研究必须接受“可验证程度分层”
本轮也再次证明：
- 有些机构页面可直接获取并引用正文；
- 有些机构只能确认存在政策入口，但暂无法直接提取全文；
- 有些机构页面链接已调整，需要重新定位。

因此研究写作中宜明确区分：
1. 已核验正文；
2. 已确认官方入口但未提取正文；
3. 待重新定位页面。

这种写法比把未核实内容直接写成结论更稳妥，也更适合后续持续补证。

## 四、对制度草案与后续研究的直接启发

### （一）对制度草案的启发
1. 可继续维持当前“授权类型分层”路线；
2. 制度之外，应同步考虑对外公开说明、FAQ 和标准表达文本；
3. 对开放资源与受限资源，应允许不同入口、不同处理路径并存；
4. 对申请人端，应逐步形成“制度 + 模板 + 说明文本”三层结构。

### （二）对下一轮研究的要求
1. 继续补充 Getty、Rijksmuseum、British Museum、National Gallery 等机构的最新公开页面；
2. 对可访问性较差的页面，优先通过替代官方入口、缓存、PDF、新闻稿或开发者/数据说明页补证；
3. 开始尝试整理国内博物馆公开可得的图片使用说明、授权流程或资源开放做法，形成国内外对照。

## 五、本轮阶段性结论
本轮进一步强化了三项判断：
1. 国外博物馆图片政策通常是“开放获取 + 受限许可 + FAQ/说明”并存的结构，而非单页式单一规则；
2. 对 `mils` 项目而言，制度建设不应只写内部条文，还应逐步建设面向申请人和使用人的对外说明文本；
3. 在研究方法上，应明确区分“已核验正文”“已确认入口”“待补核验”三种证据层级，以保证结论稳健。
