# 研究子目录说明

本目录用于沉淀：
- 研究计划与研究问题；
- 国内外案例与资料笔记；
- 授权分类、使用场景、流程与风险控制框架；
- 制度草案相关文档。

## 当前文档
- `研究范围与边界.md`：明确本项目研究对象、排除范围与制度定位；
- `研究计划.md`：阶段推进安排；
- `研究问题.md`：核心研究问题清单；
- `术语与定义（初稿）.md`：制度写作的基础术语；

- `申请材料与受理要求（研究稿）.md`：申请字段、受理判断和补正机制的研究框架；
- `审核与审批流程（研究稿）.md`：审核/审批分工、分级路径与流程链条研究框架；
- `交付方式与使用要求（研究稿）.md`：文件交付分级、使用范围与后续控制要求研究框架；
