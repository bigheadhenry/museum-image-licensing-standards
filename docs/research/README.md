# 研究子目录说明

本目录用于沉淀：
- 研究计划与研究问题；
- 国内外案例与资料笔记；
- 授权分类、使用场景、流程与风险控制框架；
- 制度草案相关文档。

- `研究计划.md`：阶段推进安排；
