# 研究子目录说明

本目录用于沉淀：
- 研究计划与研究问题；
- 国内外案例与资料笔记；
- 授权分类、使用场景、流程与风险控制框架；
- 制度草案相关文档。

## 当前文档
- `研究范围与边界.md`：明确本项目研究对象、排除范围与制度定位；
- `研究计划.md`：阶段推进安排；
- `研究问题.md`：核心研究问题清单；
- `术语与定义（初稿）.md`：制度写作的基础术语；
- `授权使用场景分类.md`：图片授权的一级使用场景划分；
- `开放获取与受限授权比较（第一轮）.md`：开放政策与受限许可的初步比较；
- `授权类型与收费减免模式（研究稿）.md`：授权类型、收费维度与减免逻辑的研究框架；
- `申请材料与受理要求（研究稿）.md`：申请字段、受理判断和补正机制的研究框架；
- `审核与审批流程（研究稿）.md`：审核/审批分工、分级路径与流程链条研究框架；
- `交付方式与使用要求（研究稿）.md`：文件交付分级、使用范围与后续控制要求研究框架；
- `署名、声明与限制条件（研究稿）.md`：署名规则、声明模板与常规限制条款研究框架；
- `风险控制与责任承担（研究稿）.md`：风险分类、责任边界、违规分级与处置机制研究框架；
- `附录模板与标准表达（研究稿）.md`：申请、审核、授权、交付、反馈与标准提示语模板体系研究框架；
- `博物馆图片授权管理制度（初稿）.md`：基于现有研究成果转写形成的制度章节初稿；
- `国内外案例线索池.md`：待持续补充的案例与公开资料入口；
- `制度草案框架.md`：制度章节结构与起草顺序建议。
