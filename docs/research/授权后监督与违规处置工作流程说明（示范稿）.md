# 授权后监督与违规处置工作流程说明（示范稿）

> 说明：本示范稿用于将馆方在图片授权作出后的持续监督、问题发现、分级判断、文书选择、升级处置、风险名单管理、后续申请控制以及复核解除等环节，整理为一条可执行的内部工作流程。该文本为内部工作说明，不直接替代具体决定文书；办理中仍应结合《整改通知单》《授权撤销或终止通知单》《风险名单记录表》《风险名单复核意见单》《后续申请强化审查意见单》《暂停受理或限制后续申请决定单》《内部违规分级判断与处置指引》等配套文本使用。

## 一、适用范围
本流程适用于以下情形：
1. 已经批准并交付的图片使用事项；
2. 已进入成果反馈、抽查、平台巡查、舆情发现或第三方投诉阶段的事项；
3. 已出现轻微偏差、一般违规、重点违规或高风险违规迹象的事项；
4. 已被纳入风险名单并需要继续管理、复核或解除的事项。

## 二、工作目标
本流程的主要目标是：
1. 把授权后监督从“偶发发现问题”转为“有固定入口的持续管理”；
2. 把问题处置从“临时反应”转为“分级判断 + 工具组合 + 文书承接”的稳定路径；
3. 把整改、声明、撤销/终止、风险名单、后续申请控制与复核解除纳入同一条可追溯链条；
4. 保持不同部门、不同承办人在类似事项上的处理口径基本一致。

## 三、建议的流程总链条
授权后监督与违规处置可按以下链条组织：

授权完成 → 交付留痕 → 成果反馈/抽查/巡查/投诉发现 → 初步核实 → 违规分级判断 → 选择处置路径 → 发出相应文书/启动相应措施 → 观察整改与持续控制 → 视情纳入风险名单或后续申请限制 → 复核、调整或解除

## 四、具体流程步骤

### 第一步：建立授权后监督入口
授权事项批准并交付后，应至少形成以下监督入口：
1. 成果反馈：要求申请主体按期提交样刊、链接、截图、样本或其他使用证明；
2. 日常抽查：对重点平台、重点合作项目、高传播事项进行抽查；
3. 巡查发现：对官网、公众号、视频平台、电商平台、素材平台等进行主动巡查；
4. 外部投诉/举报：接收馆内外对疑似违规使用的反映；
5. 舆情发现：通过舆情监测、搜索或合作单位反馈发现问题。

这一阶段的重点不是立即定性，而是保证“问题能被看见、能进入流程”。

### 第二步：初步核实事实
发现问题线索后，应先完成初步核实，包括：
1. 核对对应申请主体、授权编号、项目名称、用途、平台、期限与交付范围；
2. 确认实际使用主体与申请主体是否一致；
3. 固定初步证据，如截图、链接、视频片段、样本、页面抓取、平台记录等；
4. 判断问题是已发生、正在发生，还是仅有风险迹象；
5. 记录发现时间、发现方式和经办人。

如事实尚不清楚，可先进入补充核实，不宜仓促定性或直接升级。

### 第三步：进行违规分级判断
事实初步明确后，应结合《内部违规分级判断与处置指引》进行等级判断，重点看：
1. 行为性质；
2. 传播范围；
3. 商业化程度；
4. 是否重复发生及是否配合整改；
5. 对馆方授权秩序、公众识别与声誉的影响。

可将事项归入：
- 轻微偏差 / 可即时补正事项；
- 一般违规 / 需正式整改事项；
- 重点违规 / 需升级控制事项；
- 高风险违规 / 需直接升级与并行处置事项。

### 第四步：选择处置路径
根据分级结果，原则上可分三种路径：

#### 1. 顺序升级路径
适用于：
- 风险从低到高逐步显现；
- 主体仍有一定配合意愿；
- 事项尚未明显公共化。

通常路径：
提醒/补正 → 整改 → 名单管理/撤销终止 → 强化审查 → 复核/解除

#### 2. 并行升级路径
适用于：
- 问题已形成公共传播影响；
- 单一措施不足以控制风险；
- 既需对相对方提出要求，也需对公众划定边界。

通常路径：
整改 + 声明 + 撤销/终止评估 + 名单管理 / 后续申请限制

#### 3. 直接跳级路径
适用于：
- 明显恶意；
- 明显商业化扩散；
- 明显冒名、虚假合作、误导性背书；
- 继续放任将造成更大公共风险。

通常路径：
直接进入高风险处置组合，不再经过普通提醒或一般整改观察阶段。

### 第五步：选择并发出相应文书
根据路径和工具选择，优先使用相应文书承接：
1. 轻微偏差：记录 + 补正；
2. 一般违规：优先《整改通知单》；
3. 重点违规：同步考虑《风险名单记录表》《后续申请强化审查意见单》；
4. 高风险违规：同步考虑《对外声明与严正声明》《授权撤销或终止通知单》《暂停受理或限制后续申请决定单》；
5. 风险下降后：使用《风险名单复核意见单》进入复核、等级调整或解除路径。

文书发出时，应明确：
- 事实基础；
- 限期要求；
- 是否停止/删除/补正；
- 是否进入名单管理；
- 是否影响后续申请；
- 逾期或不配合的进一步后果。

### 第六步：跟踪整改与持续控制
文书发出后，不应视为流程结束，而应进入跟踪阶段：
1. 核查申请主体是否按要求完成整改；
2. 核查相关内容是否已删除、下架、更正或补加声明；
3. 核查是否存在继续传播、换平台传播、第三方继续使用等情形；
4. 记录整改是否真实、完整、及时；
5. 判断是否需要升级为撤销/终止、名单管理或后续申请限制。

这一阶段的重点是避免“文书发出即视为问题解决”。

### 第七步：纳入风险名单与后续申请控制
对重复违规、整改不到位、重点违规或高风险违规主体，可进入风险名单及后续申请控制路径：
1. 记录其风险等级；
2. 明确既往问题、证据和当前管理要求；
3. 视情决定是进入强化审查，还是暂停/限制后续申请；
4. 对继续提出新申请的，按强化审查或更高控制要求办理；
5. 对高风险主体，可直接限制全部或部分高风险申请入口。

### 第八步：复核、调整与解除
对已被纳入风险名单或被限制后续申请的主体，应在适当观察期后进入复核环节，重点判断：
1. 历史问题是否已实质解决；
2. 是否完成整改、履责、删除/下架或补正；
3. 观察期内是否再发生新的违规；
4. 风险是否已明显下降；
5. 是否可以维持、下调或解除其风险管理状态。

复核结果可包括：
- 维持原状态；
- 下调风险等级；
- 解除重点关注 / 移出风险名单；
- 结束限制并转入普通或强化审查路径；
- 继续维持限制状态。

## 五、部门协同建议
实际办理中，建议形成以下基本分工：
1. 承办部门：负责线索接收、事实核实、文书起草与整改跟踪；
2. 审核部门或法务支持：负责风险等级判断、工具选择和升级建议复核；
3. 审批/确认层：负责对声明、撤销/终止、名单管理、后续申请限制等重要措施作出确认；
4. 平台/宣传/技术相关部门：负责协助下架、删除、截图固定、平台沟通与证据留存。

## 六、办理中的几个关键控制点
1. **发现问题后先固定证据，再进行定性**；
2. **不要把“有反馈”直接等同于“无风险”**，仍应核查实际使用情况；
3. **不要把“发了整改通知”当成流程终点**，必须跟踪结果；
4. **已进入公共传播空间的事项，应同步考虑公众边界表达问题**，而不只处理个案相对方；
5. **对高风险主体，要单独评估继续开放申请入口是否会放大风险**；
6. **对已经限制后续申请的主体，也应保留复核与解除路径，避免无限期悬置。**

## 七、与其他内部文本的关系
本流程说明与其他文本的关系可理解为：
- 《违规情形分级与处置升级路径（研究稿）》：提供分级治理的研究逻辑；
- 《内部违规分级判断与处置指引（示范稿）》：提供馆内统一尺度；
- 本流程说明：提供实际办理顺序；
- 各类通知单、意见单、记录表与决定单：提供执行文书承接。

## 八、阶段性结论
授权后监督与违规处置，不应被理解为授权完成后的零散补救，而应视为图片授权管理制度中的持续治理链条。其核心不是“出事后怎么补”，而是：
- 让问题有入口；
- 让事实能核实；
- 让等级能判断；
- 让工具能选择；
- 让升级有路径；
- 让后续申请受控制；
- 让复核解除有出口。

只有这样，馆方的图片授权管理才能从“单次审批”进一步走向“授权前审慎判断 + 授权后持续治理”的完整制度运行方式。
