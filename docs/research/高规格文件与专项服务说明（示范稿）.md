# 高规格文件与专项服务说明（示范稿）

> 说明：本示范稿用于博物馆官方网站、数字资源平台或图片服务页面中，对高分辨率、出版级、印刷级、批量处理、特殊格式输出、加急交付等高规格文件与专项支持事项进行单独说明。其目标是进一步区分“图片可否使用”和“馆方是否提供高规格或专项服务”，防止申请人将授权批准误解为当然可取得最高质量文件或全部技术支持。正式实施时，可结合馆方技术条件、保存条件、成本结构和价格管理要求调整。

## 一、什么是高规格文件与专项服务？
本说明所称高规格文件与专项服务，主要是指超出公开页面基础下载或标准交付范围的图片文件提供与技术支持事项，包括但不限于：
1. 高分辨率图片文件；
2. 出版级、印刷级或高清展示级文件；
3. 批量文件整理、项目打包和多版本输出；
4. 文件格式转换、尺寸适配、色彩处理等专项技术服务；
5. 加急处理、特殊交付安排、补充文书或其他高工作量事项。

## 二、为什么要单独说明这类事项？
高规格文件与专项服务通常具有以下特点：
- 对文件质量、处理精度和交付稳定性要求更高；
- 对馆方人员、时间和技术资源占用更大；
- 对后续传播范围、再利用风险和控制难度影响更明显；
- 不宜当然视为基础下载或普通授权交付的一部分。

因此，即使某项使用已获得批准，馆方仍可就高规格文件或专项服务另行判断是否提供、如何提供以及是否收取相应技术/服务处理费。

## 三、常见高规格文件包括哪些？
馆方可根据实际情况，将高规格文件理解为以下一种或多种情形：
1. 分辨率显著高于网页公开版本；
2. 适用于正式出版、印刷或高清展陈的文件版本；
3. 包含特定色彩、版面或输出适配要求的文件；
4. 单次申请数量较多、需按项目统一整理的文件集合；
5. 其他明显超出一般浏览、学习、普通传播需求的文件版本。

## 四、哪些事项可能属于专项服务？
除文件本身外，以下事项也可归入专项服务：
1. 批量重命名、按项目分类或清单匹配；
2. 特定格式转换或版本适配；
3. 色彩、尺寸、版式的基础输出处理；
4. 限时、限次、指定主体的受控交付；
5. 纸质证明、附加说明、特殊交付记录；
6. 加急办理；
7. 其他经馆方认定超出标准流程的支持事项。

## 五、哪些情况下申请人可提出相关需求？
申请人可在下列情形下提出高规格文件或专项服务需求：
1. 图书、期刊、图录、学术出版等正式出版项目；
2. 展陈制作、高清屏幕展示、专题项目或大型活动；
3. 对输出质量、格式匹配或文件规格有明确要求的项目；
4. 项目有明确时间节点，需要提前安排处理和交付；
5. 其他经馆方认定可进入专项服务判断的事项。

## 六、提出此类需求时建议一并说明什么？
为便于判断，建议申请人至少说明以下事项：
1. 项目名称与用途；
2. 所需图片名称、编号、数量；
3. 所需文件规格、用途场景和输出方式；
4. 是否需要批量处理、格式转换、色彩调整或其他专项支持；
5. 是否涉及出版、印刷、高清展示、跨平台分发等高要求使用；
6. 时间节点、是否需要加急；
7. 是否存在多主体协作或复杂交付安排。

## 七、馆方通常会重点判断哪些问题？
馆方可围绕以下问题综合判断是否提供相关文件或服务：
1. 资源权利基础和授权状态是否明确；
2. 使用用途是否与申请内容一致；
3. 文件规格需求是否与实际项目需要相匹配；
4. 馆方当前是否具备相应文件版本、处理能力或交付条件；
5. 是否存在较高传播风险、再利用风险或控制难度；
6. 工作量、处理成本和时间安排是否可承受；
7. 是否需要适用额外声明、限制条件或专项协议。

## 八、哪些情况下馆方可能不同意提供？
存在以下情形之一的，馆方可不提供或暂缓提供高规格文件或专项服务：
1. 授权事项尚未获批或权利基础不清；
2. 申请用途与高规格需求不相匹配；
3. 文件规格明显超出馆方当前保存或技术处理能力；
4. 申请人未完成收费确认、协议签署或其他前置程序；
5. 涉及较高公共风险、敏感资源或后续控制困难；
6. 申请时间过于紧迫，无法在合理期限内完成处理；
7. 其他依法依规或根据业务条件不宜提供的情形。

## 九、是否一定收费？
不一定。但高规格文件与专项服务更容易涉及技术/服务处理费。馆方可根据下列因素综合判断：
- 文件规格；
- 文件数量；
- 处理复杂度；
- 时间要求；
- 交付方式；
- 是否需要附加文书或特别安排；
- 是否符合减免条件。

应特别注意：
1. 减免许可使用费，不当然意味着高规格文件或专项服务也全部免费；
2. 仅收技术/服务处理费，不等于该事项无需授权判断；
3. 馆方可根据不同项目和服务强度设置差异化处理方式。

## 十、交付时可能附加哪些控制要求？
对高规格文件与专项服务，馆方可根据风险和项目需要设置以下一种或多种控制要求：
1. 指定接收主体；
2. 受控下载链接或限时下载；
3. 限定使用平台、期限和媒介；
4. 禁止再次分发、上传素材平台或转授权；
5. 要求附加署名、声明或非背书提示；
6. 要求提交成果样本、截图、样刊或项目反馈；
7. 需要签署专项承诺或补充协议。

## 十一、为什么建议尽早提出相关需求？
高规格文件与专项服务通常比普通授权事项更依赖：
- 文件检索与准备时间；
- 技术处理时间；
- 收费与确认流程；
- 交付安排与后续控制。

如申请人临近出版、展出、上线或项目发布时才提出相关需求，馆方可能无法在短时间内完成处理。对正式出版、展陈、广告、视频制作或复杂合作项目，建议尽量提前沟通。
