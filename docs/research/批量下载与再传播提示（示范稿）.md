# 批量下载与再传播提示（示范稿）

> 说明：本示范稿用于博物馆官方网站、数字资源平台或资源下载页中，对批量下载、批量抓取、批量导出、建立本地资源库、再上传、再分发和跨平台传播等高风险行为进行专门提示。其定位不同于《下载页面提示语》：后者聚焦单次下载动作，本说明则聚焦“批量获取 + 再传播”这一更易失控的场景。正式实施时，可根据馆方技术能力、资源分级、开放政策和平台规则，将其用于批量下载页、开放 API 页面、导出工具页、接口说明页或下载协议弹窗。

## 一、适用场景
本提示主要适用于以下情形：
1. 用户拟批量下载多项图片资源；
2. 用户拟批量导出资源信息或文件包；
3. 用户拟对馆方资源进行系统性抓取、镜像保存或建立本地资料库；
4. 用户拟将已获取资源再次上传到第三方平台、素材平台、公开资源库、网盘库或合作平台；
5. 用户拟在多个平台、多个账号或多个合作链条中持续性传播相关资源。

## 二、核心边界提示
### 第一条 批量下载不当然意味着批量自由使用
本馆允许在一定条件下提供批量下载、批量导出或批量获取接口，并不当然意味着相应资源可被批量复制、批量传播、批量商用或再分发。批量获取只是文件或数据的取得方式，不当然改变各资源原有的权利状态和使用边界。

### 第二条 批量获取不当然替代逐项规则判断
即使多个资源可被集中下载或统一导出，使用者仍应根据每项资源的页面标签、开放规则、授权条件和特别说明，分别判断是否可直接使用、是否需申请、是否受特别限制以及是否可进一步传播。

### 第三条 再传播不当然包含在下载行为之中
下载、导出、缓存、同步、镜像保存或建立内部资料库，不当然意味着可将相关资源再次上传至其他平台、建立公共下载入口、提供给第三方批量取用或作为自身资源向外开放。

## 三、建议提示的重点风险
### 第四条 权利状态混同风险
批量下载页面中的资源可能并非处于完全相同的权利状态。使用者不得因资源在同一栏目、同一批次或同一压缩包中出现，就推定其均可按相同方式使用或传播。

### 第五条 传播失控风险
批量获取后的再上传、再分发、网盘共享、接口转供、素材平台挂载或合作方二次流转，容易导致传播路径失控。对传播范围大、链条长、难以追溯的事项，使用者应特别审慎。

### 第六条 平台替代风险
任何单位或个人不得将从本馆批量获取的资源，重新包装为自身可供公众检索、下载、分发或再次授权的资源库，不得以此替代本馆平台功能或造成公众对资源来源、管理主体和授权关系的误解。

### 第七条 商业化扩散风险
未经明确允许，不得将批量获取的图片资源用于商品化素材包、商用数据库、付费会员下载区、广告营销资源包或其他具有再分发、再许可、再收费性质的安排。

## 四、建议使用的简明提示语
### 版本 A
批量下载仅表示可集中获取相关文件，不当然意味着可对该批资源统一进行转载、商用、再分发或再次授权。

### 版本 B
同一批次资源的权利状态可能不同。请逐项查看资源标签、页面说明和限制条件，不要仅因可批量获取而推定可批量使用。

### 版本 C
未经明确允许，不得将批量获取的资源再次上传至第三方平台、网盘资源库、素材平台或其他公开下载环境。

### 版本 D
批量获取不当然替代正式授权。如拟进行多平台传播、合作分发、商业化使用或建立资源库，请先确认是否需另行申请。

## 五、建议设置的确认项
如页面涉及批量下载或批量导出，可考虑要求用户在下载前确认以下事项：
1. 已知晓批量获取不等于批量自由使用；
2. 已知晓同批资源可能具有不同权利状态；
3. 已知晓未经明确允许不得再上传、再分发、再授权；
4. 已知晓如拟用于正式项目、商业合作或跨平台传播，需进一步判断是否进入申请流程；
5. 已知晓如需更高规格文件、批量技术处理或专项交付安排，需另行办理服务。

## 六、建议配套显示的状态标签
批量下载、导出或接口页可结合资源类型显示以下简明标签：
1. `批量获取不等于批量使用`
2. `同批资源权利状态可能不同`
3. `禁止再上传/再分发`
4. `批量商用需另行申请`
5. `高规格批量服务需另办`

## 七、与其他文本的关系
本提示属于页面级和传播控制级说明文本，应与以下规则配套理解：
1. 《版权声明与网站使用规则》；
2. 《官网平台与数字资源公开使用边界说明》；
3. 《数字资源栏目使用提示》；
4. 《资源详情页权利标签说明》；
5. 《下载页面提示语》；
6. 《开放资源使用说明》；
7. 《图片授权申请说明页》；
8. 《图片服务办理说明》；
9. 《高规格文件与专项服务说明》。

如批量接口说明、专项协议、授权书或特别声明另有更具体要求的，以更具体文本为准。

## 八、阶段性意义
对博物馆数字资源治理而言，批量下载与再传播提示的意义在于：在“单个文件下载”之外，进一步回应平台化、规模化和再分发化使用的现实风险。它提醒使用者区分“集中获取”与“集中使用/集中传播”之间的界限，也帮助馆方在开放便利与传播控制之间建立更清晰的前端规则表达。
