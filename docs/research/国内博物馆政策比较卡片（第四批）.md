# 国内博物馆政策比较卡片（第四批）

> 说明：本文件用于继续按统一格式整理国内博物馆图片使用政策、版权声明、声明公告、联系入口与资源栏目案例，并显式标记不同对象当前的证据等级。

## 1. 湖南博物院
- 机构：湖南博物院
- 已核验程度：已核验两条官方声明/公告正文，尚未核验专门图片规则页
- 官方页面：`/zh-hans/aboutus/声明`、`/zh-hans/aboutus/【公告】严正声明`
- 本轮新增可确认规则表达：
  - 未经核实不得转载与馆方相关的不实报道；
  - 馆方免费公共资源不得被擅自用于盈利性讲解活动；
  - 未经授权利用馆方资源牟利或冒用馆方名义组织活动，馆方保留追责权利。
- 当前限制：仍未补到专门版权声明、图片使用政策或授权申请规则页
- 对 `mils` 的启发：声明/公告类文本在国内案例中可直接承担传播边界、授权边界与商业利用边界表达功能

## 本批卡片的共同观察
1. 国内馆的外部治理表达，不仅在版权声明中出现，也会分布在声明、公告、联系页和栏目结构中；
2. 对国内案例而言，声明/公告类文本在处理未经授权商业利用、冒名活动和传播误导方面尤其关键；
3. 研究时应继续坚持“已核验正文 / 已确认入口 / 已确认线索”三层以上的证据区分；
4. 这进一步支持 `mils` 把平台层说明、栏目层提示、详情页标签、下载提示、传播控制提示、接口提示、办理说明与声明/公告文本并列纳入制度原型。
