# 授权申请被暂缓、限制或不予同意情形说明（示范稿）

> 说明：本示范稿用于博物馆官方网站、数字资源平台或图片授权办理页面中的补充说明文本。其目标是帮助申请人理解：为什么部分申请不会被直接同意，为什么有的申请会被要求补充材料、缩小范围、附加条件，为什么有的事项可能被暂缓、限制或不予同意。该文本不替代正式审批决定、补正通知或个案说明，而是用于对外提前说明常见边界，减少误解和重复沟通成本。

## 一、为什么有些申请不会被直接同意？
博物馆图片授权并不是对所有申请一律作“同意 / 不同意”的简单判断，而是需要结合以下因素综合审查：
1. 图片资源的权利基础和管理基础是否明确；
2. 申请用途是否清楚、是否与馆方管理原则相符；
3. 使用范围、平台、期限、传播规模和实际使用主体是否可判断；
4. 是否存在较高的商业化、平台扩散、误导性传播、修改失真或公共影响风险；
5. 是否需要高规格文件、专项服务、专项协议或更高层级审查。

因此，部分申请可能被直接同意，部分申请可能附条件同意，部分申请可能需要补充说明、缩小范围后再判断，也有部分申请会被暂缓、限制或不予同意。

## 二、什么是“暂缓处理”或“暂缓决定”？
“暂缓”通常并不等于当然拒绝，而是表示当前材料、事实或风险判断还不足以支持馆方立即作出最终决定。

常见情形包括：
1. 申请材料中关键信息不完整；
2. 申请主体、实际使用主体、发布主体或合作方关系尚不清楚；
3. 图片用途、传播平台、使用期限、传播范围说明不足；
4. 涉及跨境传播、联名合作、重大宣传、敏感图片或复杂合作链条，需要进一步核验；
5. 需要补充样稿、视觉示意、传播方案或专项承诺；
6. 涉及权利状态、合作协议或第三方权利仍需进一步确认。

在此类情况下，馆方可能会要求申请人补充材料、修改方案或进一步说明，而不是立即给出最终结果。

## 三、哪些情况可能被要求“缩小范围”或“附加限制条件”？
有些申请并非完全不能办理，但若按申请人最初提交的方式风险较高，馆方可能要求缩小范围或附加条件后再决定是否同意。

常见情形包括：
1. 可同意在图书正文中使用，但不同意同时用于封面、海报、电商宣传页或广告投放；
2. 可同意在单一平台、单一版本、单一语种范围内使用，但不同意多平台、大范围或长期持续传播；
3. 可同意一般传播级或研究参考级文件交付，但不同意直接交付更高规格文件；
4. 可同意按既定图像使用，但不同意大幅裁剪、重组、包装设计或再制作；
5. 可同意用于特定非营利项目，但不同意进一步转为商业宣传、品牌联名或产品营销；
6. 可同意在整改、补充声明或承诺后继续推进，但不同意无条件直接放行。

这类“限制”并不当然意味着否定项目，而是馆方为控制风险所采取的边界管理方式。

## 四、哪些事项更容易进入强化审查或更严格控制？
以下事项通常更可能进入强化审查、附加条件控制或更高层级判断：
1. 商业广告、品牌推广、联名合作；
2. 涉及大规模出版、电商传播、平台分发或多渠道商业传播；
3. 涉及跨境传播、多语言发布或境外平台使用；
4. 涉及敏感文物图片、特殊保护要求图片或权利状态复杂资源；
5. 涉及明显改动、封面/主视觉使用、广告主图使用；
6. 申请主体、实际使用主体、发布主体、合作方之间关系复杂；
7. 可能引发误导性背书、公共争议或较大声誉风险的事项。

对于上述事项，馆方可能要求更完整的材料、更具体的样稿、更强的声明和限制条件，或者进入专项审查程序。

## 五、哪些情况下馆方可能不予同意？
存在以下情形之一的，馆方可能不予同意：
1. 图片权利基础或管理基础不清；
2. 涉及第三方权利但未完成权利清理；
3. 申请用途明显超出馆方管理原则；
4. 申请事项存在较高公共风险、敏感风险或后续控制难度过高；
5. 申请人提供虚假信息、隐瞒关键事实或补正后仍不足以判断；
6. 申请事项明显可能导致公众误认为馆方对项目、活动、产品或观点作出认可或背书；
7. 申请方案存在明显低俗化、戏谑化、失真性包装或其他不当表达风险；
8. 其他依法依规不宜授权的情形。

不予同意并不意味着馆方否定所有后续申请，而是表示在该次申请条件、用途或风险结构下，不宜提供授权。

## 六、哪些主体或事项可能被“限制后续申请”？
馆方在特定情况下，可能对部分主体的后续申请采取更严格控制，甚至暂停受理或限制某类事项申请。

常见情形包括：
1. 曾出现超范围使用、擅自商业化扩散、未按要求署名、擅自修改或擅自再传播；
2. 曾因虚假申报、拒不整改、拒不删除下架而被纳入风险名单；
3. 虽已整改，但观察期内风险仍未明显下降；
4. 同类问题重复发生，显示其信用和合规能力存在持续疑问；
5. 历史问题尚未实质解决，但再次提出新的高风险申请。

此类情况下，馆方可能要求：
- 提交更完整说明；
- 提交专项承诺；
- 接受强化审查；
- 缩小申请范围；
- 暂缓、限制或不予同意。

## 七、“不予同意”是否等于以后都不能再申请？
不一定。

很多情况下，不予同意是基于当次申请的用途、传播结构、合作模式、风险程度或材料情况作出的判断，并不当然表示申请人今后任何事项都不能再申请。

但如申请人存在以下情形，其后续申请可能继续受到更严格控制：
1. 历史违规问题尚未实质整改；
2. 多次出现信用或合规问题；
3. 再次申请的事项与此前高风险情形高度相似；
4. 虽更换项目名称，但实际传播模式和风险结构并无明显变化。

## 八、申请人如何降低被暂缓、限制或不予同意的可能性？
申请人可通过以下方式提高办理效率并降低误解风险：
1. 如实说明申请主体、实际使用主体、发布主体和合作方关系；
2. 清楚说明用途、平台、期限、语种、区域、传播规模和项目时间安排；
3. 如涉及商业传播、联名合作、跨境传播、封面/主视觉使用、明显修改或多平台传播，应主动提前说明；
4. 需要时主动提交样稿、视觉示意、传播方案、合同关系说明或项目证明；
5. 不将“可下载”“公开展示”“学术用途”“公益项目”等理解为当然无需审批或当然免费；
6. 如有既往申请或整改记录，应如实说明相关情况并提交整改完成证明。

## 九、如果馆方要求补充材料、缩小范围或附加条件，应如何理解？
这通常表示：
- 馆方认为该事项并非当然不能办理；
- 但按当前方案直接同意风险较高或条件不够明确；
- 通过补充材料、缩小范围或附加条件，有可能使事项进入可控范围。

因此，申请人如收到此类要求，建议重点理解以下问题：
1. 哪些信息仍不清楚；
2. 哪些传播范围、使用形式或合作安排需要进一步限定；
3. 哪些声明、署名、成果反馈或交付条件是馆方特别关注的；
4. 是否需要重新调整项目方案后再继续申请。
