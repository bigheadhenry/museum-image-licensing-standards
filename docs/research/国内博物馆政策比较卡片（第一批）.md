# 国内博物馆政策比较卡片（第一批）

> 说明：本文件用于按统一格式整理国内博物馆图片使用政策、版权声明与授权入口案例，便于与国外案例并行比较。

## 1. 故宫博物院
- 机构：故宫博物院
- 已核验程度：已核验正文
- 官方页面：影像授权 / 版权声明
- 可确认结构：版权声明 + 影像授权申请入口并存
- 可确认规则：拥有版权的影像资源可授权用于论文、出版、展览、宣传等明确用途；商业使用须书面许可；转载/引用/下载原则上需注明来源；未经许可不得修改
- 申请路径：下载申请函后按要求提交材料至指定邮箱/传真；区分大陆与港澳台及海外联系路径
- 对 `mils` 的启发：国内馆常以网站版权声明作为基础规则，再通过专门授权入口处理个案申请

## 2. 中国国家博物馆
- 机构：中国国家博物馆
- 已核验程度：已核验正文
- 官方页面：版权声明
- 可确认结构：以网站版权声明为主要公开规则页
- 可确认规则：未经书面许可不得商业使用；转载/引用/摘编/下载需注明来源；未经许可不得修改；进一步使用应联系馆方相关部门
- 风险与责任边界：保留侵权通知、不保证声明、第三方链接不构成认可和免责声明
- 对 `mils` 的启发：国内馆网站版权声明本身已承担初步分流和约束功能，可视为制度外部接口的一部分

## 3. 上海博物馆
- 机构：上海博物馆
- 已核验程度：已确认官方入口，正文待补提取
- 官方入口线索：站内 `privacy` / `copyright` 页面返回 200
- 当前状态：自动提取未稳定拿到正文
- 对 `mils` 的启发：国内案例研究需接受“先确认官方入口，再逐步补正文”的工作方法

## 4. 南京博物院
- 机构：南京博物院
- 已核验程度：已确认官方入口，正文待补提取
- 官方入口线索：站内 `zh-CN/copyright` / `zh-CN/privacy` 页面返回 200
- 当前状态：自动提取未稳定拿到正文
- 对 `mils` 的启发：国内馆公开规则可能嵌在站点框架中，后续需要更强的页面提取手段

## 本批卡片的共同观察
1. 国内案例比国外更常见“版权声明先行、正式授权另行联系”的结构；
2. “商业用途须书面许可”“注明来源”“不得擅自修改”是高频基础规则；
3. 网站版权声明虽然不等于完整制度，但在实际治理中已承担基础约束与分流作用；
4. 后续如要形成更贴近国内落地的制度原型，应补一层“版权声明/网站使用规则”文本，而不仅是授权制度和表单模板。
