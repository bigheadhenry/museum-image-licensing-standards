# 国内博物馆图片使用政策与版权声明比较（第三轮）

## 一、本轮目标
在前两轮已经完成故宫博物院、中国国家博物馆的正文核验，并对上海博物馆、南京博物院、陕西历史博物馆、湖南博物院等对象形成分层证据结构后，本轮继续推进国内案例补证，重点放在：
1. 继续寻找国内馆官网中与版权、声明、使用规则、数字资源栏目相关的真实官方页面；
2. 将“可直接引用的正文”“可稳定确认的官方入口”“可确认的平台/栏目结构”继续区分开来；
3. 补强国内案例的分层研究方法，使 `mils` 对国内博物馆图片治理结构的理解更稳健。

## 二、本轮补证对象与结果

### 1. 湖南博物院（从“待继续补证”提升为“已核验官方声明正文”）
本轮核验页面：
- `https://www.hnmuseum.com/zh-hans/aboutus/声明`
- `https://www.hnmuseum.com/zh-hans`

本轮可确认要点：
1. 湖南博物院中文官网首页可稳定返回 HTTP 200；
2. 官网首页可明确展示其线上展览、精品文物等数字展示属性，说明其官网具有数字资源与线上展示功能；
3. 官方页面《声明》可稳定访问，并可直接提取正文；
4. 该《声明》正文明确出现以下信息：
   - 有外部媒体发布了与湖南博物院相关的不实报道；
   - 湖南博物院明确指出该新闻报道并非其官方正式发布，部分情况不属实；
   - 如涉及相关事项，应以湖南博物院官方网站正式对外发布信息为准；
   - 希望相关媒体及机构停止转载并删除不实报道；
   - 以后如进行与湖南博物院相关的采访报道，应与湖南博物院核实后再进行对外发布。

本轮阶段性判断：
- 虽然本次拿到的并不是专门的“版权声明”或“图片使用规则”页面，而是一则官方《声明》；
- 但该页面具有重要的制度意义，因为它明确反映出：湖南博物院对外信息发布与转载行为具有“官方发布优先、未经核实不得转载”的治理意识；
- 因此，湖南博物院可以从此前“待继续补证”的状态，提升为“已核验官方声明正文，但尚未核验专门图片规则页”。

制度启发：
- 国内馆的外部规则表达并不总是集中为一页“图片使用政策”；
- 有时，声明、公告、官网正式发布机制本身，也反映了机构对信息转载、权威来源和传播控制的基本态度；
- 这提示 `mils` 在国内案例研究中，不应只盯住“版权声明”四个字，也要关注具有规范功能的官方声明、公告与说明页。

### 2. 苏州博物馆（首次补入：已稳定确认官网与馆藏栏目官方入口）
本轮核验页面：
- `https://www.szmuseum.com/`
- `https://www.szmuseum.com/Collection/Index`
- `https://www.szmuseum.com/Wap/Home/`
- `https://www.szmuseum.com/Wap/Collection/Index`

本轮可确认要点：
1. 苏州博物馆官网首页、馆藏栏目页、移动端首页和移动端馆藏栏目页均可稳定返回 HTTP 200；
2. 其官网主站与移动端站点均表现为统一官方站点结构，而非无关页面或失效入口；
3. `Collection/Index` 与 `Wap/Collection/Index` 的存在，至少可稳定证明苏州博物馆官网具有面向公众的馆藏/资源栏目入口；
4. 当前获取结果虽受响应头兼容问题影响，尚未稳定抽出完整正文，但“官网 + 馆藏栏目”这一结构已经可以被确认。

本轮仍未确认的事项：
1. 尚未定位到明确的版权声明页、图片使用规则页或授权申请页；
2. 尚不能断言其是否公开区分一般网站内容规则、数字资源规则与图片个案授权路径。

本轮阶段性判断：
- 苏州博物馆可作为新的国内对象纳入线索池；
- 当前证据等级适合归入“已确认官方入口/栏目结构，但未见专门规则页正文”的层级；
- 它的当前价值不在于条文细节，而在于补强了“国内馆官网常以栏目结构呈现资源访问，而非直接提供统一图片政策页”的理解。

制度启发：
- 对国内馆而言，馆藏栏目本身常是公众接触数字资源的第一入口；
- 因而制度原型不能只写抽象规则，还要继续重视栏目提示、详情页标签、下载提示等页面层表达。

## 三、本轮比较后的国内阶段性判断

### 1. 国内案例的“规范性文本”不一定只表现为版权声明
湖南博物院本轮最重要的收获，不是拿到了一页经典意义上的“版权规则”，而是拿到了一页具有明确规范功能的官方《声明》。这表明：
- 国内馆的外部治理表达，可能分散在版权声明、公告、声明、官网说明、栏目提示等不同页面类型中；
- 研究时若只寻找“Image Licensing”或“Copyright Policy”式的单一页面，容易漏掉国内治理实践的真实样貌。

### 2. 栏目/资源入口结构本身就是制度外部接口的一部分
苏州博物馆本轮虽未拿到完整规则页，但已稳定确认：
- 官网主站存在；
- 馆藏栏目存在；
- 移动端对应栏目也存在。

这说明：
- 国内博物馆即使没有一页集中式公开制度，也常常已经通过官网/馆藏栏目把资源公开访问结构建起来；
- 对 `mils` 而言，这进一步支持“平台层说明 + 栏目提示 + 详情页标签 + 下载提示 + 办理说明”的多层原型路径。

### 3. 国内案例证据等级需要继续细分
经过三轮补证，国内案例的证据层次可以更清楚地理解为：
1. **已核验正文（专门规则或声明正文）**：如故宫、国博、本轮的湖南博物院《声明》；
2. **已确认官方入口但正文待补提取**：如上海博物馆、南京博物院；
3. **已确认官网平台/栏目结构或页脚权利信息，但未见专门规则页**：如陕西历史博物馆、本轮的苏州博物馆；
4. **待继续补证**：仍需进一步寻找更明确页面的对象。

这比单纯区分“已核验/未核验”更符合国内实践，也更利于后续制度草案稳健引用案例。

## 四、对 `mils` 的直接启发
1. 国内案例研究应把“官方声明/公告类规范文本”纳入有效证据范围，而不是只寻找专门版权规则页；
2. 对国内数字资源治理而言，“官网正式发布机制”“转载前核实要求”“栏目入口结构”都具有制度意义；
3. `mils` 当前已经形成的平台层、栏目层、详情页层、下载层、批量传播层、接口层和办理层文本，恰好与国内馆这种分层表达现实更为贴近；
4. 下一步若继续补国内案例，可优先寻找：
   - 苏州博物馆是否存在页脚版权、声明或使用条款入口；
   - 湖南博物院是否另有版权、隐私、资源使用说明页面；
   - 其他地方馆是否存在“声明/公告 + 数字资源栏目 + 联系办理”式组合结构。

## 五、本轮阶段性结论
第三轮国内案例研究虽然仍未新增大量“完整规则正文”，但补入了两个重要方向：
- 湖南博物院：已拿到可直接引用的官方《声明》正文，说明国内馆的治理表达可能通过声明/公告而不仅是版权页体现；
- 苏州博物馆：已稳定确认官网与馆藏栏目官方入口，进一步支持“栏目结构本身也是公开治理接口”的判断。

这使 `mils` 对国内博物馆图片治理的理解更进一步：国内实践往往不是以一页完整的“图片使用政策”出现，而是通过官网正式发布、栏目结构、一般权利边界与个案办理路径共同构成分层治理体系。
