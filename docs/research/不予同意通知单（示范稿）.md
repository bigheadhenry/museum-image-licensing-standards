## 二、处理结果

## 三、不予同意原因
本次不予同意，通常系基于以下一种或多种情形（可勾选或填写）：
- 图片权利基础或管理基础当前不清；
- 涉及第三方权利、合作协议限制或其他未完成权利清理事项；
- 申请用途明显超出馆方管理原则或不符合本馆授权边界；
- 申请事项存在较高的公共传播风险、声誉风险或误导性背书风险；
- 涉及商业广告、品牌联名、平台分发、重大商业化传播等高风险用途，且现有条件下风险不可控；
- 涉及敏感图片、限制公开资源或其他不宜授权事项；
- 申请主体、实际使用主体、发布主体或合作方关系复杂，且经核实仍无法形成清晰可控的授权边界；
- 申请材料存在虚假、隐瞒、重大不一致，或补正后仍不足以支持判断；
- 申请方案涉及明显不当修改、失真表达、低俗化包装或其他不宜支持的传播方式；
- 即使缩小范围、附加条件或进一步限制，风险仍难以降至可接受水平；

## 四、馆方判断说明
馆方作出本次不予同意决定，主要基于以下判断逻辑：
1. 在当前申请事项、用途、传播方式、合作结构及风险条件下，授权边界难以明确或难以有效控制；
2. 如在当前条件下作出授权，可能对馆方管理秩序、资源边界、公众理解或后续监督造成明显困难；
3. 本次不予同意系针对当前申请事项所作出的具体判断，并不当然等同于对申请主体今后所有申请事项一律否定。

## 五、后续处理要求
请贵方在收到本通知后注意以下事项：
1. 不得以“已申请”“曾沟通”“正在研究”或其他类似理由擅自使用相关图片；
2. 如贵方已基于本次申请准备样稿、页面、物料或传播安排，不得将其视为已获馆方许可；
3. 如贵方已在未获授权前先行上线、发布、展示、传播或交付第三方，应立即停止相关使用，并根据馆方要求删除、下架或整改；
4. 如贵方拟就相关事项另行调整方案、缩小范围、变更用途或补充新的事实基础，应重新判断是否另行提交新申请；
5. 如本次不予同意系基于权利基础不清、第三方权利未处理、合作结构不明或高风险传播安排等事项，贵方应先行解决相应问题后，再视情决定是否重新申请。

## 六、是否可重新申请
本次不予同意，并不当然排除贵方在下列条件变化后另行提出新申请：
1. 使用用途、传播范围、合作模式、发布平台或成果形式已实质调整；
2. 风险结构明显下降；
3. 已补充新的权利说明、合作说明、样稿、证明文件或专项承诺；
4. 原导致不予同意的核心问题已被实质解决；
5. 馆方认为具备重新进入受理与审核程序的基础。

但如存在以下情形，贵方的后续申请仍可能继续受到更严格控制：
- 曾提供虚假材料或隐瞒关键事实；
- 曾未经授权先行使用相关图片；
- 曾拒不配合补正、整改或删除下架；
- 重新申请事项与本次高风险结构高度相似，且无实质变化；
- 其他显示信用或合规能力仍存在明显问题的情形。

## 七、与其他处理措施的关系
1. 本通知适用于“本次申请不予同意”的正式结果告知；
2. 如事项仅属材料不足、事实待补充，原则上应优先使用《补正通知单》或《暂缓决定告知单》；
3. 如贵方已发生未授权先行使用、拒不整改、冒名活动、商业化扩散或其他违规行为，馆方可在本通知之外，另行启动整改通知、撤销/终止、对外声明、风险名单或限制后续申请等措施；
4. 本通知本身不替代其他责任追究措施。

## 十、使用说明（馆内）
1. 本通知适用于“已形成明确不予同意判断”的事项，不宜用暂缓决定长期替代；
2. 应尽量写明不予同意的核心理由及后续是否存在重新申请可能，避免外部误解为笼统拒绝；
3. 如不予同意同时伴随未授权先行使用、虚假申报、拒不整改等情形，可联动启动其他处置文书；
4. 本通知应与申请编号、审核意见、补正记录、暂缓记录（如有）、审批结论及留档材料对应保存。
