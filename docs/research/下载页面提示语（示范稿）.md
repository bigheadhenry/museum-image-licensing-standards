# 下载页面提示语（示范稿）

> 说明：本示范稿用于博物馆官方网站、数字资源平台或图片资源下载页中的简明提示文本，重点用于在用户点击下载、获取链接、导出文件或接收压缩包之前，对下载行为的含义、使用边界、后续义务和风险提示作出直接说明。其定位不同于《数字资源栏目使用提示》与《资源详情页权利标签说明》：前两者分别偏栏目层和详情页层，本说明则聚焦“下载动作发生时”的最后一道前端提示。正式实施时，可根据页面长度、按钮布局、文件规格与风险等级简化为短提示、弹窗说明或确认勾选语。

## 一、核心提示文本（长版）
您当前下载的是本馆提供的特定版本数字资源文件。请注意：
1. 文件可下载，不当然意味着可被任意复制、转载、再传播、商业使用或再次授权；
2. 文件可获取，不当然意味着可用于超出页面说明、开放规则或授权条件之外的用途；
3. 即使页面允许下载，使用者仍应遵守来源标注、声明要求、非背书、禁止再授权和其他限制条件；
4. 如需将该资源用于出版、展览、视频制作、宣传传播、商业合作、批量分发、跨平台传播或其他正式项目，请进一步确认是否需进入授权申请或服务办理流程；
5. 如需高分辨率、出版级、印刷级、批量整理、格式转换或加急交付等支持，请按本馆相关服务规则另行办理。

## 二、核心提示文本（短版）
### 版本 A
下载文件不等于取得无限制使用权。请按页面规则、来源标注要求和相关授权条件使用。

### 版本 B
本次下载仅表示文件获取，不当然包含转载、商用、再传播、再授权或高规格服务支持。

### 版本 C
如拟正式项目使用、高规格输出或跨平台传播，请先确认是否需申请授权或另行办理服务。

## 三、建议配套显示的下载状态提示
下载页面可结合资源状态或文件类型，配套显示以下一项或多项简明标签：
1. `浏览版下载`：表示当前仅提供浏览或一般参考版本；
2. `开放版下载`：表示当前文件可在公开规则范围内直接使用，但仍受相应条件约束；
3. `授权后交付`：表示需申请获批后方可获取文件；
4. `高规格文件需另办`：表示更高规格文件不在当前下载范围内；
5. `下载不等于商用许可`：提示下载动作不当然等于已获商业使用资格。

## 四、下载前建议提示的最小问题集
下载页面的提示语，至少应帮助用户在下载前再次确认以下问题：
1. 当前下载的是什么版本文件；
2. 当前下载是否等于可直接使用；
3. 是否仍需署名、声明或遵守限制条件；
4. 是否需另行申请正式项目使用；
5. 是否需另行办理高规格文件或专项服务；
6. 是否存在第三方权利或外部风险需自行判断。

## 五、下载页面常见误解及建议对应提示
### （一）误解：能下载 = 能随便用
建议提示：`可下载仅表示可获取当前文件版本，不当然意味着可任意转载、传播、商用或再次授权。`

### （二）误解：能下载 = 已获得馆方许可
建议提示：`下载行为不当然替代正式授权。对出版、展览、视频、商业合作等事项，请先确认是否需申请。`

### （三）误解：能下载 = 能拿到最高规格文件
建议提示：`当前下载文件不当然为高分辨率或出版级版本。如需更高规格文件，请查看服务办理说明。`

### （四）误解：能下载 = 可上传到其他平台再分发
建议提示：`未经明确允许，不得将下载文件上传至素材平台、开放下载平台、网盘资源库或其他第三方平台再次分发。`

## 六、可用于弹窗/勾选框的确认语示例
### 示例 1
我已知晓：本次下载仅表示获取当前文件版本，不当然取得无限制使用权；后续使用仍须遵守页面规则、来源标注要求及相关授权条件。

### 示例 2
我已知晓：如拟将所下载文件用于出版、展览、视频、宣传、商业合作或其他正式项目，应进一步确认是否需申请授权或另行办理服务。

### 示例 3
我已知晓：未经明确允许，不得将下载文件再次上传、批量分发、转授权或用于使公众误认为馆方背书的情形。

## 七、与其他文本的关系
下载页面提示语属于最接近“文件实际离开平台”时的一层前端表达，应与以下文本配套理解：
1. 《版权声明与网站使用规则》；
2. 《官网平台与数字资源公开使用边界说明》；
3. 《数字资源栏目使用提示》；
4. 《资源详情页权利标签说明》；
5. 《开放资源使用说明》；
6. 《图片授权申请说明页》；
7. 《图片服务办理说明》；
8. 《高规格文件与专项服务说明》。

如资源详情页、开放规则、授权书或专项协议另有更具体要求的，以更具体的文本为准。

## 八、阶段性意义
对博物馆数字资源治理而言，下载页面提示语的意义在于：把“文件可被获取”与“文件可被如何使用”明确区分开来，在最容易发生误解的动作节点上，重新提醒用户理解权利边界、申请分流和服务分流。这一层虽短，但对降低误用、减少争议和控制传播风险具有直接作用。
