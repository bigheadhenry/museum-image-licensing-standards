## 二、处理结果
经对贵方提交的图片授权申请及相关补充材料进行审查，现就本次申请事项作出如下处理结果：

- **本次申请同意**；
- 馆方同意贵方在后续完成收费/减免确认、授权文书生成及文件交付等必要程序后，按核准范围使用相关图片；
- 贵方应按照本通知及后续文书载明的用途、期限、平台、署名、声明和其他要求使用图片。

## 四、后续办理事项
本次同意后，原则上仍应按下列顺序继续办理：
1. 完成收费或减免确认；
2. 生成或签发《图片授权书》；
3. 完成交付前核验与文件交付；
4. 按要求提交成果反馈。

如涉及不收费事项，也不当然免除授权书、交付确认、署名声明与成果反馈等后续要求。

## 五、申请主体应注意的事项
1. 本次同意不等于可在未完成后续程序前先行使用；
2. 未完成收费确认、授权书签发或文件交付前，不得主张已取得完整使用权；
3. 应按馆方要求署名、标注来源并附必要声明；
4. 不得超出核准用途、平台、期限、地域和数量使用图片；
5. 不得擅自修改、再传播、转授权或上传至素材平台、开放下载平台或其他未经核准的渠道；
6. 使用完成后，应按要求提交链接、截图、样刊、样书、视频片段或其他成果反馈材料。

## 六、与后续文书的关系
1. 本通知用于确认“同意”这一审批结果；
2. 具体费用安排，以《收费通知与减免确认单》为准；
3. 具体授权条件，以《图片授权书》为准；
4. 实际交付文件、接收主体和文件规格，以《图片交付记录单》为准；
5. 使用后的留档与复核，以《成果反馈表》及相关监督记录为准。

## 七、违反核准范围或后续要求的后果
如贵方存在下列情形之一，馆方可视情采取补正、整改、停止使用、删除下架、补缴情形费用、撤销/终止授权、纳入风险名单或限制后续申请等措施：
1. 超出核准用途、平台、期限、地域或数量使用；
2. 擅自修改、再传播、转授权或交由未核准第三方使用；
3. 未按要求署名、附声明或提交成果反馈；
4. 其他违反授权条件或管理要求的行为。

## 十、使用说明（馆内）
1. 本通知适用于已形成明确“同意”判断，且事项将进入收费确认、授权书、交付和反馈链条的情形；
2. 如事项虽原则上可同意，但仍需通过平台、期限、文件、修改、声明等条件强化控制，原则上应优先使用《附条件同意通知单》；
3. 本通知不宜替代《收费通知与减免确认单》《图片授权书》或《图片交付记录单》本身；
4. 本通知应与申请编号、审核意见、审批结论及后续执行文书对应保存。
