# 开放资源使用说明（示范稿）

> 说明：本示范稿用于面向公众说明：哪些博物馆图片资源属于可直接使用的开放资源、开放使用并不等于毫无条件、哪些事项仍需进入正式授权流程。其定位不同于《版权声明与网站使用规则》：后者主要处理网站内容的一般边界，本说明则聚焦“已开放图片资源”的直接使用规则。正式实施时，可结合馆方开放政策、资源分级、页面标签体系和法务意见调整。

## 一、总则
### 第一条 适用对象
本说明适用于本馆通过官方网站、数字资源平台、专题页面或其他公开入口明确标注为可开放使用的图片资源。

### 第二条 文件目的
制定本说明，旨在帮助使用者区分“可直接使用的开放资源”与“仍需正式申请的受限资源”，明确开放资源的使用边界、来源标注、禁止事项和风险提示要求。

### 第三条 基本认识
开放资源，是指本馆已通过公开规则明确允许在一定范围内直接使用的图片资源。开放资源的存在，不当然意味着：
1. 所有馆藏图片均已开放；
2. 所有开放图片均无条件使用；
3. 所有文件规格、服务内容和处理支持均可免费获取；
4. 使用者可据此主张本馆对相关项目、产品、活动或观点作出认可与背书。

## 二、开放资源的识别
### 第四条 识别方式
使用者应根据本馆公开页面、资源详情页、下载页或统一说明中的标签、声明、授权说明、权利状态说明或特别提示，判断图片资源是否属于开放资源。

### 第五条 非开放资源的基本判断
存在以下情形之一的，原则上不得仅依据本说明直接使用，而应进一步判断是否需要正式申请：
1. 页面未明确标注开放使用规则；
2. 权利状态不清或存在限制提示；
3. 涉及第三方权利、敏感内容或合作限制；
4. 页面明确要求联系馆方、提交申请或另行取得许可；
5. 使用者无法合理判断是否属于开放范围。

## 三、开放资源的可直接使用规则
### 第六条 可直接使用的基本情形
对已明确标注为开放资源的图片，使用者可按照页面说明直接进行浏览、下载、引用、展示、研究、教学传播及其他规则允许的使用，无需逐案申请。

### 第七条 来源标注要求
除本馆明确豁免外，使用开放资源时，原则上仍应保留必要来源信息。来源标注可包括但不限于：
1. 馆名；
2. 图片名称或藏品名称；
3. 图片编号、资源编号或藏品编号；
4. 本馆指定的统一署名语；
5. 页面链接或资源来源说明。

### 第八条 版本与文件规格说明
开放资源的直接使用，通常仅限本馆公开提供的现有文件版本。开放资源可开放基础下载，并不当然意味着本馆必须提供：
1. 更高分辨率文件；
2. 出版级、印刷级或无压缩版本；
3. 色彩处理、去底、抠图或版式适配；
4. 加急支持、纸质证明、重新摄影或其他专项服务。

如使用者需要上述支持，应按本馆正式程序另行联系或申请。

## 四、开放资源的使用边界
### 第九条 不得误导公众
使用开放资源时，不得以任何方式使公众误认为本馆对相关项目、产品、活动、宣传内容、合作安排或观点作出认可、推荐、联合发布或其他形式的背书。

### 第十条 不得超越特别限制
即使属于开放资源，如页面、标签或配套说明中另有特别限制，使用者仍应遵守。例如：
1. 需保留特定权利声明；
2. 不得删除来源信息；
3. 不得用于特定高风险传播场景；
4. 不得上传至第三方素材平台或再许可平台；
5. 不得将资源包装为自身可再次授权的内容。

### 第十一条 修改与再制作边界
对开放资源是否允许裁剪、拼接、调色、覆盖文字、再设计、再编辑或作为二次创作素材使用，应以本馆页面说明或配套规则为准。即使允许一定程度的再利用，也不得：
1. 伪造资源来源；
2. 歪曲藏品原意或历史语境；
3. 损害文物形象、馆方声誉或公共利益；
4. 制造本馆认可、联名或合作的误导印象。

## 五、何时仍需正式申请
### 第十二条 需要转入正式授权流程的情形
即使相关图片属于开放资源，存在以下情形之一的，使用者仍应联系本馆或按正式程序提交申请：
1. 需要高规格文件、专项处理或个性化服务；
2. 拟用于商业出版、广告营销、品牌联名、商品包装或其他高敏商业场景；
3. 拟在大规模平台分发、跨境传播或复杂合作链条中使用；
4. 拟对图片进行较大幅度改动、再包装或嵌入复杂产品；
5. 页面规则已明确要求进入申请程序；
6. 其他本馆认为需个案判断的情形。

### 第十三条 开放使用与正式授权的关系
开放资源制度的目的，在于降低对低风险、已明确开放事项的重复审批成本；但开放使用并不替代本馆对高风险、复杂用途、专项服务和特殊场景的正式授权管理。

## 六、第三方权利与风险提示
### 第十四条 第三方权利提示
即使本馆将某些图片资源列为开放资源，也不当然对所有可能涉及的第三方版权、商标权、肖像权、隐私权、公开权、合同限制或其他外部权利风险作出无限保证。使用者应结合实际用途自行审慎判断，并承担相应责任。

### 第十五条 使用者责任
使用者应自行确保其实际使用方式符合适用法律法规、本说明、本馆公开页面说明及其他相关规则，不得以“开放资源”为由当然主张对所有用途均可免责使用。

## 七、违规处理
### 第十六条 违规情形
以下行为可视为违反本说明：
1. 删除或伪造来源信息；
2. 擅自超越页面规则或特别限制使用；
3. 误导公众认为本馆作出认可或背书；
4. 将开放资源再包装为可由使用者再次授权或收费分发的资源；
5. 其他违反本馆公开规则或法律法规的行为。

### 第十七条 处理措施
对违反本说明的行为，本馆可视情采取下列一种或多种措施：
1. 要求停止使用、删除、下架或更正；
2. 要求补充来源标注或补作说明；
3. 要求转入正式授权程序或补办手续；
4. 对严重情形限制其后续获取资源或服务；
5. 依法追究相应责任。

## 八、附则
### 第十八条 与其他文本的关系
本说明与本馆《版权声明与网站使用规则》、图片授权制度、个案授权书、开放声明或其他特别说明不一致的，以后者中更具体、更明确的规则为准。

### 第十九条 迭代完善
本说明为示范性文本，后续可根据馆方开放资源范围、页面标识方式、咨询高频问题和实际使用争议持续修订完善。
