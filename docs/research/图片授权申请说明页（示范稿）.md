# 图片授权申请说明页（示范稿）

> 说明：本示范稿用于博物馆官方网站、数字资源平台或图片服务页面中的“申请说明”入口文本。其目标不是替代《图片使用申请表》或《图片授权书》，而是帮助外部申请人快速判断：是否需要申请、谁可以申请、需准备什么材料、提交后会经历什么流程、多久能获得反馈，以及哪些高风险事项可能触发进一步审查。正式实施时，可结合馆方平台结构、办理渠道、价格政策与法务意见调整。

## 一、哪些情形需要提出图片授权申请？
存在以下情形之一的，原则上应向本馆提出图片授权申请：
1. 拟将图片用于图书、期刊、图录、电子出版物等出版项目；
2. 拟将图片用于展览展示、宣传传播、视频节目、纪录片、课程教材等项目；
3. 拟将图片用于广告营销、品牌合作、商业传播、商品包装或其他商业用途；
4. 拟获取高分辨率文件、出版级文件或其他专项服务；
5. 拟对图片进行裁剪、拼接、局部放大、叠加设计元素或其他再制作；
6. 页面未明确标注为开放资源，或使用者无法判断是否可直接使用；
7. 本馆认为应进入正式申请程序的其他情形。

## 二、哪些情形可能无需逐项申请？
对已由本馆通过公开页面、开放声明或统一规则明确标注为可直接使用的开放资源，使用者可按照相关公开规则直接使用，无需再逐项申请。

但即使属于开放资源，如存在以下情况，仍建议联系本馆或提交申请：
1. 需要更高规格文件或专项服务；
2. 拟用于商业出版、广告营销、品牌合作等高敏场景；
3. 拟进行较大幅度修改、再包装或复杂平台传播；
4. 页面规则对具体用途存在保留或限制。

## 三、谁可以提出申请？
下列主体可根据实际需要提出图片授权申请：
- 个人研究者、作者、学生；
- 高校、科研机构、文化机构；
- 出版机构、媒体机构；
- 企业、社会组织或其他项目主体；
- 经合法授权代为办理申请的代理人或合作方。

如申请主体与实际使用主体、实际发布主体、制作主体不一致，应在申请中如实说明相关关系。

## 四、申请前建议准备哪些材料？
为提高办理效率，建议申请前准备以下信息或材料：

### （一）主体信息
- 申请主体名称；
- 联系人姓名、职务、联系方式；
- 身份证明或单位证明；
- 如为代理申请，提供委托关系说明。

### （二）项目与用途说明
- 项目名称；
- 使用目的；
- 成果形式；
- 是否营利；
- 是否涉及发行、销售、赞助、联名、广告或平台分发；
- 预计发布时间、展出时间或上线时间；
- 传播平台、受众范围、使用地域、使用期限。

### （三）图片需求信息
- 藏品名称、图片名称或资源编号；
- 图片数量；
- 每张图片拟使用的位置或用途；
- 所需文件规格；
- 是否需要高分辨率文件、特殊处理、重新扫描或重新拍摄。

### （四）修改与再利用说明
- 是否拟裁剪、拼接、局部放大；
- 是否拟叠加文字、动画、特效或包装设计元素；
- 是否拟作为封面、主视觉、广告主画面使用；
- 是否拟在多个平台重复传播；
- 是否拟由第三方代为制作、发布或运营；
- 是否拟再许可、上传素材库或二次分发。

### （五）减免或特殊情形证明（如适用）
- 学术研究、教育教学、公益传播、公共文化服务等相关说明；
- 立项通知、课程计划、论文录用通知、合作函、研究计划等支持材料；
- 其他有助于说明减免理由或特殊情况的证明。

## 五、如何提交申请？
本馆可根据实际情况提供线上表单、电子邮件、系统申报或纸质材料等申请渠道。申请人应按页面公布方式提交完整材料，并确保信息真实、准确、完整。

如本馆已提供统一《图片使用申请表》，建议优先使用统一表单提交申请。

## 六、申请提交后会经历哪些环节？
图片授权事项原则上按照以下链条办理：
1. 申请提交；
2. 形式受理；
3. 材料补正（如需）；
4. 实质审核；
5. 审批决定；
6. 费用确认或减免确认（如适用）；
7. 授权文书生成；
8. 文件交付与留档。

其中：
- **受理**主要判断材料是否完整、信息是否足以判断；
- **审核**主要判断权利基础、使用场景、授权类型、收费适用与风险事项；
- **审批**主要决定是否同意、是否附条件同意、是否暂缓或不同意。

## 七、馆方可能要求补正哪些内容？
如申请材料存在以下问题，本馆可能要求补充或更正：
1. 申请主体信息不完整；
2. 用途说明不清楚；
3. 图片对象无法准确识别；
4. 使用媒介、范围、期限、传播平台等核心信息缺失；
5. 是否商业使用或是否存在合作安排说明不足；
6. 修改、再传播、第三方参与情况说明不足；
7. 主张减免但未提交必要证明材料。

申请人应在通知期限内完成补正；逾期未补正或补正后仍不足以判断的，本馆可不继续办理。

## 八、哪些事项可能进入强化审查？
以下事项可能触发更高层级或更严格的审查：
1. 商业广告、品牌营销、联名合作；
2. 大规模出版、平台分发或多渠道商业传播；
3. 涉及敏感文物图片、特殊合作协议资源或权利状态复杂资源；
4. 涉及明显改动、再制作、封面/主视觉/广告主图使用；
5. 申请主体、实际使用主体、发布主体不一致；
6. 可能引发声誉、法律或公共影响风险的其他事项。

## 九、多久可以获得反馈？
本馆可根据事项复杂程度设置一般处理时限。普通事项通常在材料齐备后按顺序办理；涉及补正、高风险事项、联合审查、专项服务或权利状态需进一步核验的，处理时间可能相应延长。

如申请事项时间敏感，建议申请人尽早提交，并在申请中说明项目时间节点。

## 十、是否一定收费？
不是。是否收费以及收费方式，应结合以下因素综合判断：
- 授权类型；
- 使用场景；
- 是否营利或存在商业安排；
- 使用范围、期限、平台和传播规模；
- 文件规格与技术处理工作量；
- 是否符合减免条件。

部分事项可不收费，部分事项可仅收技术/服务处理费，部分事项则需收取许可使用费或其他合理费用。

## 十一、减免申请如何处理？
如申请事项属于学术研究、教育教学、公益传播、公共文化服务或其他具有明显公共价值的情形，申请人可按要求提出减免申请，并提交相关证明材料。

减免是否适用，由本馆结合项目用途、风险程度和服务成本综合判断。减免并不等于无条件使用，也不免除署名、声明、成果反馈和风险控制要求。

## 十二、获批后需要注意什么？
经批准取得图片后，申请人应当：
1. 仅在核准用途、范围、期限、媒介和平台内使用；
2. 按要求署名、标注来源并附必要声明；
3. 不得擅自修改、再传播、转授权、转售或另作他用；
4. 按要求提交成果样本、样刊、链接、截图或其他反馈材料；
5. 不得使公众误认为本馆对相关项目、活动、产品或观点作出认可或背书。

## 十三、哪些情况下可能不予授权？
存在以下情形之一的，本馆可不予授权或暂缓处理：
1. 权利基础不清；
2. 涉及第三方权利未清理完成；
3. 申请用途明显超出馆方管理原则；
4. 存在较高公共风险、敏感风险或后续控制难度过高；
5. 申请材料虚假、关键事实隐瞒或补正后仍不足以判断；
6. 其他依法依规不宜授权的情形。

## 十四、违规使用会如何处理？
对超范围使用、未按要求署名、擅自修改、擅自再传播、虚假申报、擅自商业化扩散等行为，本馆可视情要求：
- 更正或补正；
- 停止使用、删除、下架；
- 补办授权或补缴情形费用；
- 撤销或终止授权；
- 限制后续申请；
- 依法追究相应责任。

## 十五、其他说明
本说明为示范性文本，正式实施时可与《常见问题说明》《版权声明与网站使用规则》《开放资源使用说明》配套使用，共同构成博物馆图片授权管理制度的对外说明体系。
