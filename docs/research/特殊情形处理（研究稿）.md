# 特殊情形处理（研究稿）

## 一、本轮目标
在制度初稿已基本覆盖一般授权流程、收费减免、交付使用、风险控制和模板体系的基础上，本稿进一步研究博物馆图片授权中的“特殊情形处理”问题。

本稿重点在于明确：
- 哪些事项不宜直接纳入普通授权流程；
- 对跨境传播、敏感文物图片、重大合作项目和争议性用途，应如何提高审查强度；
- 如何把特殊情形处理从“临时拍脑袋决定”转化为制度化路径。

## 二、基本判断：特殊情形不是“例外杂项”，而是高风险事项的制度入口
图片授权制度如果只覆盖常规场景，而将高风险事项完全留给临时判断，会导致：
- 标准路径与高风险路径混杂；
- 经办人无明确升级标准；
- 审批层级和责任边界不清；
- 同类事项处理尺度不一致。

因此，特殊情形处理应被理解为：当事项超出常规授权边界时，制度为其预设的升级审查、附加控制和决策留痕机制。

## 三、建议纳入特殊情形范围的事项

### （一）跨境传播事项
通常包括：
- 在境外出版物、网站、平台、社交媒体或数据库中使用；
- 面向境外公众持续传播；
- 涉及多语言页面、多地区同步发布；
- 由境外主体、境外合作方或跨境平台参与发布、制作、运营。

该类事项的主要风险在于：
- 传播路径较长，后续控制和下架纠偏难度更高；
- 不同法域、不同平台规则下的限制执行难度更高；
- 图片可能被再次转载、聚合、素材化扩散。

### （二）敏感文物图片事项
通常包括：
- 具有明显敏感属性、特殊保护要求或公开限制要求的文物图片；
- 涉及重要细节、特殊视角、尚不宜公开信息的图片；
- 可能引发文化误读、历史误读、宗教伦理争议或公共舆情风险的图片。

该类事项的主要风险在于：
- 不当传播可能引发较大公共影响；
- 一旦扩散，馆方难以及时收回；
- 对图像修改、语境包装和商业化利用应特别审慎。

### （三）重大合作项目
通常包括：
- 与大型出版机构、平台、媒体、品牌方或商业机构的集中合作；
- 多批次、多资源、长期性授权安排；
- 可能对外形成示范案例、品牌联动或持续传播效应的合作。

该类事项的主要风险在于：
- 授权不是单一图片、单次使用，而是可能演变为长期合作关系；
- 合作结构复杂，实际控制主体和传播链条可能较长；
- 若条件不清，容易被外界理解为馆方背书或深度合作代言。

### （四）争议性或高敏传播用途
通常包括：
- 广告营销、品牌联名、热点事件借势传播；
- 明显娱乐化、低俗化、戏谑化包装；
- 可能引发重大价值争议、公共情绪反弹或声誉损害的传播用途。

该类事项的主要风险在于：
- 对馆方和文物形象造成直接影响；
- 容易突破一般的“已获授权即可使用”理解边界；
- 需要更明确的价值判断和更高层级把关。

## 四、特殊情形的建议处理路径

### （一）启动条件
出现以下任一情形时，宜进入特殊情形处理路径：
1. 审核人判断事项超出常规授权尺度；
2. 权利基础虽可判断，但传播外溢和后续控制难度明显升高；
3. 涉及敏感文物图片、重大商业合作、跨境平台传播或公众争议风险；
4. 申请主体、实际使用主体、合作方之间关系复杂；
5. 需要额外声明、专项协议或更高层级审批。

### （二）处理步骤
特殊情形处理可考虑至少包括以下步骤：
1. 书面标记为特殊情形事项；
2. 补充收集合作结构、传播路径、使用样稿或详细方案；
3. 进行专项风险说明；
4. 必要时启动联合审核或集体审议；
5. 将附加限制条件明确写入审批决定和授权文本；
6. 对交付方式、接收主体、成果反馈和后续监督设置更严格要求。

### （三）可能的处理结果
特殊情形处理结果不应只有“同意/不同意”两种，还可包括：
- 同意，但附加强限制条件；
- 要求补充方案、样稿或承诺后再决定；
- 缩小使用范围、期限或平台后附条件同意；
- 暂缓决定；
- 不同意。

## 五、特殊情形中的重点控制要求

### （一）跨境传播控制要求
1. 明确核准平台、核准语种、核准地区和核准期限；
2. 明确禁止再次上传至其他平台、数据库或素材库；
3. 必要时要求更强的来源标注和限制提示语；
4. 可要求使用低于常规商业传播的交付规格；
5. 可要求提交上线链接、页面截图和阶段性反馈。

### （二）敏感文物图片控制要求
1. 明确是否允许公开展示、局部展示或仅限内部/受控使用；
2. 严格限制裁剪、重组、商业化包装和戏剧化呈现；
3. 必要时要求逐图审批，而非按项目打包审批；
4. 可要求增加更高层级审批或专项说明；
5. 对公开传播范围和素材留存方式实行更严控制。

### （三）重大合作项目控制要求
1. 明确合作各方身份、角色和责任分工；
2. 区分项目主办方、制作方、发布方和运营方；
3. 明确是否允许阶段性复用、批次追加或系列传播；
4. 明确馆方不作商业背书的边界；
5. 必要时通过专项协议而非简单授权书处理。

### （四）争议性用途控制要求
1. 要求申请主体提交完整传播方案、样稿或视觉示意；
2. 明确禁止误导、低俗化、歪曲性和价值失当表达；
3. 对外部包装语、标题、封面文案和视觉组合可提出前置审查要求；
4. 发现明显争议风险时，可直接不予授权。

## 六、与制度其他章节的衔接
特殊情形处理不应被孤立，而应与以下部分联动：
- 与“风险控制与责任承担”衔接：特殊情形本质上是高风险事项的制度化入口；
- 与“审核与审批流程”衔接：应明确何时升级审查、何时集体审议；
- 与“交付方式与使用要求”衔接：高风险事项不应按常规规格和常规交付方式处理；
- 与“署名、声明与限制条件”衔接：特殊情形通常需要增强版声明与限制语；
- 与“模板体系”衔接：应通过整改通知、撤销/终止通知、交付记录等模板增强可执行性。

## 七、可直接转化为制度条文的表达方向
1. 对跨境传播、敏感文物图片、重大合作项目、争议性用途等特殊情形，馆方可启动专项审查、联合审核或集体审议程序；
2. 特殊情形事项应根据具体风险增加附加条件、限制性要求和更严格的交付控制；
3. 对传播路径复杂、后续控制难度较高或存在较大公共影响风险的事项，馆方可要求补充材料、样稿、承诺或专项协议；
4. 对不宜控制风险或明显不符合馆方管理原则的事项，馆方可不予授权。

## 八、本轮阶段性结论
本轮研究形成五项可直接支撑制度完善的结论：
1. 特殊情形处理不是零散例外，而是高风险事项的升级审查机制；
2. 跨境传播、敏感文物图片、重大合作项目和争议性用途应优先纳入特殊情形范围；
3. 特殊情形的关键不只是“更高层审批”，还包括更严格的材料要求、交付控制和后续反馈；
4. 对高风险特殊事项，制度应允许“缩小范围后同意”“补充材料后再决定”“专项协议处理”等中间结果；
5. 特殊情形章节可以成为制度初稿从常规流程迈向更成熟治理结构的重要补充。
